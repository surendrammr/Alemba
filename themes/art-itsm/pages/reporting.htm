Alemba Service Manager provides easy to use designer applications to build and compose interactive, real-time BI dashboards and powerful reports.

Manage and share dashboards and reports with other analysts and users.
Harness the power of the Alemba API to create easy access to the Alemba Service Manager data and custom fields.
Built-in security – users viewing dashboards can only see permitted data.

Dashboards

  • Over 30+ different data visualisation widgets and user filters.
  • Custom widgets can be added to display data how you want to.
  • Drag and drop designers and data source creation.
  • Data drill down and linking to other dashboards.
  • Easily integrate dashboards within Alemba Service Manager, Self Service Portal and Nano.
  • Mobile management BI applications for iOS and Android.
  • Effortless export to many standard file formats: Excel, PDF, Images, etc.
  • ETL data preparation, manipulation and use of expression columns.
  • Built-in collaboration and comments.

Reporting

  • View and edit standard RDL and RDLC reports.
  • Rich selection of report items such as charts, data bars, tables, pivot tables, barcodes and maps.
  • Drag and drop designers and data source creation.
  • Easily integrate reports within Alemba Service Manager, Self Service Portal and Nano.
  • Effortlessly print and export reports to PDF, Word, Excel, CSV, PowerPoint and HTML.
  • ETL data preparation, manipulation and use of expression columns.
  • Built-in collaboration and comments.
