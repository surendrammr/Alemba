Synonyms

With the right permissions, you can maintain a list of synonyms that are defined for a keyword. You can attach keywords to Knowledge Bank articles and use the keyword or synonym to search for articles.

Adding a Synonym

Before you start

You must have Text Retrieval Setup selected in your own General Access security role before you can create synonyms.

  1. Select ≡ > Admin > System Administration.The System Administration window appears.

  2. In the Explorer pane, under Text Retrieval, select Synonyms.

    A browse table appears, with all of the keywords on the system with synonyms.

  3. Select . The Select Keyword dialog box appears with a list of all the keywords that have been defined for your system.
  4. At Keyword, select the keyword for which you want to define a synonym. The entry is highlighted.
  5. Select Ok. The dialog box closes and the selected keywords are added to the Keyword list on the Synonyms window. Key the synonym for the keywords in the Synonym list. You can adjust the column widths if required.
  6. Repeat to add more synonyms. You can select the same keyword again if you wish to have several synonyms for the same keyword.
  7. Select to save the changes. Provide the Change Reasons if prompted to do so. The list is redisplayed, with your new synonyms listed in uppercase.

Deleting a Synonym

Before you start

You must have Text Retrieval Setup selected in your own General Access security role before you can delete synonyms.

You cannot reverse this procedure. If you deleted a word by mistake, you have to re-create the entry by adding the word to the list.

  1. Select ≡ > Admin > System Administration.The System Administration window appears.
  2. In the Explorer pane, under Text Retrieval, select Synonyms.
  3. In the browse table, select the synonym you want to delete and then select .
  4. The entry disappears, but the changes are not permanent until you save the changes.
  5. Repeat for each synonym you want to delete.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so. The list is redisplayed and the changes are permanent.
