CMDB Service Levels

You can define different service levels that Analysts can assign to CMDB items as part of the CMDB Item details.

Creating a Service Level

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select CMDB Service Levels to open the window .
  1. Select . A new row is created to the browse table.
  2. In the Service Level list, type the name of the new service level you want to create.
  3. Repeat for each service level you want to create.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

The order in which the service types display in the drop-down list on the CMDB Item Details window matches the order in which they appear on the table. Select a service and use or to change its position in the table.

Renaming a Service Level

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select CMDB Service Levels to open the window .
  1. Select the service level entry you want to rename.
  2. Type the name you want to assign to the selected service.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a Service Level

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

You cannot reverse this procedure. If you delete a service by mistake, you have to re-create the entry by creating a new service.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select CMDB Service Levels to open the window .
  1. Select the service level entry you want to delete from the list by selecting on it.
  2. Select . If you attempt to delete the system default, you will receive a warning message.The entry is deleted.
