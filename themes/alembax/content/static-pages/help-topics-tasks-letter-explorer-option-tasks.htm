Letter Explorer Option

You can create letters (documents) directly from the Call Details, Task Details and Request Details windows, using applications such as Microsoft Word and attach those letters to the call/task/request. The letters are pre-formatted, containing details from the current task/request.

Creating a Letter

  1. In the Call/Task/Request Details window, select the Letter explorer option to display the Attach Letter to Call/Task/Request window with a list of current templates and existing letters. The letters automatically contain information from the ASM System and can be configured for your organization. You can select and modify one of the existing letters or create a new one based on one of the templates. ASM will only show letters specific to ASM. From the templates list, select the template on which you want your document to be based.
  2. Type the name you want to call your new document in the Create a name for selected template field. This is a mandatory field.
  3. Click Create Letter. Microsoft Word opens with the document you specified, based on the selected template. All relevant data will be inserted into the document from ASM.
  4. Make any necessary changes and when you are satisfied with the document, save it and exit Microsoft Word in the normal way. You will automatically be brought back to ASM.
  5. Click Back on the Attach Letter to Task window to return to the Task Details window.

Opening an attached letter

  1. In the Created/Attached Letters list, select the letter you want to view.
  2. Click Open.

Deleting an attached letter

  1. In the Created/Attached Letters list, select the letter you want to delete.
  2. Click Delete and confirm the deletion.
