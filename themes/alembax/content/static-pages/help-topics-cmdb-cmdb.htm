CMDB

ASM Core allows you to create and manage the following CMDB Items:

  • Service
  • Service Action
  • Service Bundle
  • Configuration Item, including contracts, cost centers, jurisdictions, locations and organizations
  • Software Product
  • Inventory
  • Assets
  • Stakeholders
  • Subscriber Groups
  • Transactions
  • Service Portfolio

You can link items so that you can anticipate the impact of outages and plan upgrades.
