Screen Set Explorer Option

The Screen Set explorer option allows you to associate another screen set with the current call or call template.

This option is only enabled if:

  • You are using multiple call screens
  • Enable Call Screen Set on Explorer is selected on the IPK Settings window
  • The system administrator has defined custom call screen sets in IPK administration
  • The system administrator has created custom screens in ASM Designer
  • You have access to more than one call screen set in the Screen Sets tab of your IPK management security role.

The Screen Set option enables you to do the following:

  • If you are viewing a Call, it allows you to choose a custom call screen set and thus use another Call Details window, that is, the one associated with the screen set you select.
  • If you are working with a Call Template, it allows you to choose a custom call screen set, and thus the default set of call screens for this screen set. In this way, you can also associate a custom portal submission form with a call template. Users will complete the submission form linked to this screen set if they log a call through Self Service Portal. Users may for example order a service through Self Service Portal Service Request Catalog and thus use (such as a service order for a service action or service bundle) through Self Service Portal.
  1. Display the Call Details window if it is not already on screen.
  2. In the explorer pane, expand Workflow, then select Screen Set.
  3. In the New Call dialog box, select the screen set you want to apply. The currently applied screen set is not displayed.Â 
  4. Click OK. The screen refreshes and displays the Details window belonging to the selected screen set.

You can use the ASM Designer to configure call screens, if you have the required security role permissions.
