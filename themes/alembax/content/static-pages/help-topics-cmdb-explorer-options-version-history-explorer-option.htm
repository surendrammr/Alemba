Version History Explorer Option - CMDB Item

The Version History explorer option enables you to view the history for the CMDB item. The history includes relevant changes to each version of the CMDB item. Note that new versions are only created when you clone a CMDB item; not when you copy it.

  1. Display the <CMDB Entity> Details window , if it is not already on screen, by searching for the CMDB item.
  2. From the CMDB Item Details explorer pane, expand the Data tab if necessary and select Version History .
  3. The <Entity> Version window appears, with a browse table showing the list of versions. You can adjust the column widths if required.
  4. To view more information on a particular version of the CMDB item, select the entry and select . (This is grayed out if there is only one version of the item.) The CMDB Item Details window appears for the selected entry. This window displays information about the config item, its version, the date on which the version was created and the name of the analyst who created the new version.
