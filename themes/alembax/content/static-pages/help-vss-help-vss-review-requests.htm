Reviewing Requests in Self Service Portal

In Self Service Portal, you can search for and view requests that have been completed or implemented, and those that are open and pending implementation or completion.

You must have Review Own Requests, Review Org Requests and/or Review All Requests selected in the Options tab of your Self Service Portal role, depending on your viewing requirements.

  1. From the My Options menu, select Requests, and then Review Past/Current Requests.
  2. If you are looking for a specific request, type its number in the Request No field. Otherwise leave this field blank.
  3. Select the Request State.

    4. Open Requests to search for and return results on requests that are currently open
    Resolved < 1 Week to search for and return results on requests that were implemented and completed in the last week
    Resolved < 4 Weeks to search for and return results on all requests that were implemented and completed in the last 4 weeks
    Not Submitted to search for and display requests for which you have completed or partially completed a submission page, but have not submitted the page yet
  4. Click to display the Request Summary.
  5. The requests are listed in a browse table. Select any one of the requests by double clicking on it. This will display the Review Past/Current Requests page, in which you can view the details, as well as add further details, notes and attachments, and submit the request, if required and you have the correct permissions.

Troubleshooting

No Search Results Displayed

If your search does not return any results, modify search criteria to make it more general.
Too Many Search Results Displayed

If your search returns too many results, refine your search by providing more information about the request/s for which you are searching.
