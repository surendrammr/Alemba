Reviewing Configuration Items

A configuration item (or config item, or just CI) is any resource that needs to be identified in the day to day activities of a business. It can be defined as 'anything that can break or fail, which has a value'. By this definition a config item can be a PC, a telephone, a software system, or any service or product that needs to be supported. Configuration items can also included warranties, agreements, documentations, contracts, etc. Your system may require analysts to identify a config item when logging a call.

Self Service Portal enables you to search for and view the details of configuration items that have been enabled with Portal access.

  1. In the My Options menu, select the Configuration Items option. Then select Review Configuration Items.
  2. The Review Configuration Items page appears. Complete as many of the details as necessary.

    3. Your Config Items Select this option to search for and return results on config items that are enabled for request or active against your user role
    Organization Config Items Select this option to search for and return results on config items that are enabled and in use for your organization. If you are not linked to an organization, this option will not be displayed.
    All Config Items Select this option to search for and return results on all config items where the permission to view has been enabled for your user role. This could include config items that are enabled and active for other organizations or users.
    CI TItle Specify a configuration item title if you wish.
    CI Ref If you are searching for a specific config item and you know the config item reference, type it here.
    CI Type Use the drop-down list to select a configuration item type.
  3. Click OK to view the search results.
  4. Your search results are displayed in a browse table on the Configuration Item Summary page.
  5. To view one item in detail, select it. It appears in the Review Config Items page.
  6. You can attach files to the configuration item from the details page.
