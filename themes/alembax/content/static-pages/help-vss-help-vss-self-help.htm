Self Service Portal Self Help

Self Service Portal's Self Help is available from the My Options menu.

It offers the following options:

  • Search Knowledge Bank, which enables you search for information within the Knowledge Bank to locate articles that contain information about types of Incidents and Requests
  • Self Diagnosis, which enables you to use scripts to assist you with identifying solutions for common issues
  • FAQ, which enables you to find information on common problems and suggested solutions
  • Known Issues, which enables you to search for Known Issues to find a resolution to your Problem before logging an Incident
  • Ask a Question, which enables you to submit a query to a service desk operator
  • View Your Questions, which enables you to view questions you have submitted
