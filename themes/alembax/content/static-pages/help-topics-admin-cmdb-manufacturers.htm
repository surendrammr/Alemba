Manufacturers

You can define a list of manufacturers which can be assigned to configuration items in the CMDB.

You may define as many manufacturers as you wish.

Creating a Manufacturer

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Manufacturers to open the window. Existing manufacturers are displayed in alphabetical order.
  1. Select . A new row is added to the browse table.
  2. In the empty row, type the name of the manufacturer you want to add to the Manufacturers list.
  3. Repeat for each manufacturer you want to create.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Once you select , the browse table displays the entries in alphabetical order.

Renaming a Manufacturer

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Manufacturers to open the window. Existing manufacturers are displayed in alphabetical order.
  1. Select the manufacturer you want to rename.
  2. Overtype with the new name you want to assign to the selected manufacturer.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a Manufacturer

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

You cannot reverse this procedure. If you delete a Manufacturer by mistake, you have to re-create the entry by creating a new manufacturer type.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Manufacturers to open the window. Existing manufacturers are displayed in alphabetical order.
  1. Select the manufacturer you want to delete.
  2. Select . If you attempt to delete the system default, you will receive a warning message.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
