Viewing Event Types

Before you start

You must have Integration Setup selected in the Admin tab of your General Access Security Role.

  1. Select â‰¡ > Admin > Integration.
  2. In the Explorer pane, under Integration, select Events.
  3. Select a source from the Filter by Source list to view all the event types discovered on the selected source by a connector with event management capability.

    4. If the source does not have any events defined, a pop-up message appears to this effect.

    If ASM encounters an error connecting to the selected source (such as a connection or configuration error), an error pop-up message appears.

    The Mapped checkbox on the Integration Events window indicates whether mappings have been defined for the corresponding event type.
