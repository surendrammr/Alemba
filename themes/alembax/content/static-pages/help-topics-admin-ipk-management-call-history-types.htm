Call History Types

The call history types are displayed in a call’s history. These are defined on the system. You cannot create your own call history type, but you can update the descriptive name to clarify its use.

Renaming Call History Types

Before you start

As some parameters are read from the database at application start-up, we recommend that there are no Analysts logged on when you make changes to system settings.

Ensure you are in the correct partition before applying the settings.

You must have IPK Set Up selected in the Admin tab of your General Access Security Role to configure the IPK management settings.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand IPK Management.

  2. Select Call History Types to open the window. A browse table displays all of the history types and their descriptions . You can adjust the column widths if required. Scroll down to see them all.
  3. Select the call history type you wish to rename, and in the Name column, overtype the existing name with the new one.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.
