Dashboard platform installation overview

Install and configure the dashboard platform and its components in the following order:

  1. Install the dashboard platform on either a single server or multiple servers.

    For more information, see Installing the dashboard platform.

  2. Configure the Dashboard Server.

    For more information, see Configuring the Dashboard Server.

  3. Configure the User Management Server.

    For more information, see Configuring the User Management Server.

  4. If required, configure external access to the Dashboard Server.

    For more information, see Configuring external access to the Dashboard Server.

  5. If you use Active Directory to authenticate and authorize your users in a Windows domain type network, then enable the User Management Server for Active Directory.

    For more information, see Enabling User Management Server for Active Directory authentication.

  6. Connect ASM to the User Management Server.

    For more information, see Connecting Alemba Service Manager™ to the User Management Server.

  7. Connect ASM to the Dashboard Server.

    For more information, see Connecting Alemba Service Manager™ to the Dashboard Server.

  8. Configure the Alemba API client for dashboards.

    For more information, see Configuring the Alemba API connection.

  9. Install Alemba Dashboard Designer.

    For more information, see Installing Alemba Dashboard Designer.

  10. Connect Alemba Dashboard Designer to the Dashboard Server.

    For more information, see Connecting Alemba Dashboard Designer to the Dashboard Server.

  11. For information about creating dashboards using the Web Designer, see Creating dashboards using the designers.
  12. For information about configuring the ASM dashboard menu to open specific dashboards, see Displaying a dashboard in ASM.
