PowerShell requirements

Windows Management Framework includes PowerShell 5.0 or above, which you can install on earlier Windows Server versions.

Consult the Microsoft online help to identify and download the package you require for your OS platform.

Identifying the PowerShell version

If PowerShell already exists on your server, you can identify which version you are running:

  1. Start a PowerShell session (type PowerShell into your Windows search bar, then select the app).

  2. At the command prompt, type Get-Host.

    The version number appears, along with the rest of the app information.
