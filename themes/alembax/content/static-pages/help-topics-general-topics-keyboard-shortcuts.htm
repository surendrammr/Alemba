Keyboard Shortcuts

ASM Core supports the following shortcut keys to display commonly used windows from anywhere within the application.

F8 On the Call Details window, navigate between the Actions & Solutions tab and the History tab.
F9 To show/hide the explorer on any window
F7 To perform a spell check.

In addition to the shortcuts listed above, ASM Core provides a wide range of shortcuts for all the options available on the menu items.

Some shortcuts are assigned to more than one action (for example, Ctrl T is assigned for opening the Tasks Outstanding and Timesheet window). If you add both the actions to your list of Buttons with Shortcuts, one of the options will be automatically reassigned a different shortcut.

You can create more keyboard shortcuts as part of personalizing your system.
