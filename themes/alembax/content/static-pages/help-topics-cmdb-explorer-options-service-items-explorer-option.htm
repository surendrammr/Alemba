Service Items Explorer Option

You can view all the Services, Service Actions, and Service Bundles linked to the location.

  1. Display the Location Details window, if it is not already on screen.

    Search for the location, select it from the browse table of results and then select .

  2. In the Explorer pane, expand Location, then select Service Items.
  3. The Linked Service Items window appears, with a browse table listing all service items linked to the location. Select to make the buttons beneath the table active if necessary.
  4. To view the details of a linked service item, select the item and then select Open.
  5. To link a service item, select . The CMDB Item Search window appears pre-selected for a service item search (that is, services, service actions, and service bundles). Search for and link the relevant service items.
  6. To remove a linked service item, select the item in the browse table and select Remove.
