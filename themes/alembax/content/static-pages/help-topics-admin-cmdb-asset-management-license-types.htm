Asset Management - License Type

You can define different license types for Analysts to use when adding the details of Transactions made against Software Products.

Creating a License Type

Before you start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings. Depending on if/how your system is partitioned, ensure that you are in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Asset Management - License Type to open the window. The window displays the current license types.
  1. Select . A new row is added to the browse table.
  2. In the blank row, type the name or description of the type of license you with to define.
  3. Repeat for each license type you want to create.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Renaming a License Type

Before you start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings. Depending on if/how your system is partitioned, ensure that you are in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Asset Management - License Type to open the window. The window displays the current license types.
  1. Select the license type entry you want to rename by selecting on it.
  2. Type the new name you want to assign for the license type.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a License Type

Before you start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings. Depending on if/how your system is partitioned, ensure that you are in the correct partition.

You cannot reverse this procedure. If you delete an entry by mistake, you have to re-create the entry by adding a new license type.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Asset Management - License Type to open the window. The window displays the current license types.
  1. Select the license type entry you want to delete from the list.
  2. Select . If you attempt to delete the system default, you will receive a warning message.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
