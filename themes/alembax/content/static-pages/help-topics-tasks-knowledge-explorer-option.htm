Knowledge Explorer Option - Tasks

You can search for and link knowledge bank entries to the current request through the current task. All knowledge entries are attached at the level of the request. Therefore, entries linked from a task will actually link to the request itself (and be visible from all tasks in the request).

  1. On the Task Details window, in the Explorer pane, select Knowledge. The Linked Knowledge window appears with a list of any linked knowledge bank articles, if there are any. You can also create knowledge entries from this window.
  2. To link an article, click . The Knowledge Search window appears.
  3. Search for and select the article(s) that you want to link to the current task.
  4. To unlink an entry, select the linked entry in the browse table of the Linked Knowledge Entries window, and click . If you are unable to find any existing knowledge entries to link to the current task, you may want to create a new entry and link it to the request. You can do so from the Linked Knowledge window.
  5. Click . The New Knowledge Entry window appears.
  6. At Type, select the entry type you want to create.
  7. Click OK. The Knowledge Entry Details window appears. This window will contain different fields based on the entry type you selected.
  8. Complete the fields and save your changes.
  9. The Knowledge Entry Details is closed and the Linked Knowledge window appears with the entry you created linked to the current task.
