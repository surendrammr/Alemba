Support Skills Explorer Option

You can view a list of analysts with specific areas of expertise while in a Call Details window. This is useful if you need to forward the call to another analyst (or group) with the necessary skills to resolve the call.

The Support Skills options in the IPK Settings (Partitioned) window determine how support analysts are profiled. They can be profiled on Type and/or CMDB Type. Skills are assigned to individual analysts in the Skills explorer option on the Person Details window.

  1. Display the Call Details window if it is not already on screen.
  2. In the explorer pane, expand Resolution, then select Support Skills.
  3. In the Support Skills window, all analysts whose skills profile matches that of the currently selected Type Tiers or CMDB Item Types are listed.
  4. To change the display of analysts based on either the Type (the Type Tiers) or CMDB Item Types, select the relevant option option.
  5. To view the details of any analyst listed, highlight their name. Full details appear in the Job Title, Telephone, and Technical Background fields.
  6. Once you have identified the analyst to whom you intend to forward the call, highlight their name in the Analysts box.
  7. Click OK. The current call is forwarded to the selected analyst.

If your ASM System has IPK Workflow Rules enabled and a workflow rule for assigning a call is detected, the rule will take precedence over support skills for assigning the call to an analyst.
