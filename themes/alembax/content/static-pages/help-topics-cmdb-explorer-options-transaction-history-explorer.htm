Transaction History Explorer Option

You can search on transactions added to a software product or inventory. Every time a new transaction is performed on an inventory or software product, a new record is added to the transaction history.

  1. Display the Transaction History window from the Software Product details window.
    • Display the software product window , if it is not already on screen, by searching for the CMDB item and specifying the CMDB Type as Configuration Item\Software Product.
    • If required, select to activate the toolbar buttons.
    • In the Explorer pane, expand Licenses, then select Transaction History.

    2. or

    Display the Transaction History window from the Inventory Product details window.
    • Display the inventory item window , if it is not already on screen, by searching for the CMDB item and specifying the CMDB Type as Configuration Item\Software Product.
    • If required, select button to activate the toolbar buttons.
    • In the Explorer pane, expand Bulk Inventory, then select Transaction History.

    or

    Display the Transaction History window from a Call.
    • Display the Call details window , if it is not already on screen.
    • If required, select to activate the toolbar buttons.
    • In the Explorer pane, expand Data, then select Transaction History.
  2. The Transaction History window appears. This window contains filter options for the software product or inventory details and a browse table with a list of the transactions added to the software product or inventory.
  3. Select and specify the details by which you want to filter the transaction records.
