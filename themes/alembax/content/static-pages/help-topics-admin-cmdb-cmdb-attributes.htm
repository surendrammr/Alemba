CMDB Attributes

You can define categories of information related to CMDB Items, Person Details, External Suppliers, Organizations, Locations or Contracts which are not recorded in the main details window. Analysts can then view and specify information in these categories by selecting the Attributes explorer option for the specific entity in the CMDB.

Creating an Attribute

Before You Start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings. If your system is partitioned, ensure that you are working in the correct partition.

Select the Attributes checkbox in System Settings to see these options in the Explorer pane.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select one of the following menu options:
  • CMDB Item Attributes
  • Contract Attributes
  • Person Attributes
  • Location Attributes
  • Organization Attributes
  1. Select Add in the Tier 1 table to add a value to the topmost tier .
  2. In the blank row, type the name of the attribute.
  3. To add lower tiers linked to the first tier, select the attribute in the Tier 1 table, and then select Add in the Tier 2 table.
  4. In the blank row, type the name of the tier 2 attribute.
  5. Repeat to create Tier 3 attributes.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so.
  7. Repeat to create other attributes, saving as you go.

Deleting an Attribute

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

You cannot reverse this procedure. If you delete an attribute category by mistake, you have to recreate the entry. If you delete an attribute, all tiers below the attribute are also deleted. Details already specified for deleted category are retained, but Analysts cannot specify new values for this category.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select one of the following menu options:
  • CMDB Item Attributes
  • Contract Attributes
  • Person Attributes
  • Location Attributes
  • Organization Attributes
  1. Select the attribute you want to delete. Remember, to display attributes in Tiers 2 or 3, you must select the relevant attribute in Tier 1.
  2. Select Delete.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
