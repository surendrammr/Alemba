Outages Explorer Option

The Outages Explorer option enables you to view the outages planned for any CMDB items linked to the request.

Before you start

You must have Create Outage from Request selected in your Availability Management Security Role.

  1. Select the Take Action button on the Request Details window if necessary to allow you to work on the request.
  2. Select the Outages option from the Data group.

    3. If an outage is planned to run against any of the CMDB items linked to the request, a small orange square appears next to the option.

  3. The Request Outages window appears, listing existing outages created by the current request for any linked CMDBÂ items.
  4. To update the details of an outage, select the row and click Open.

  5. The Outage Details window appears with details of the outage for the selected row. The browse table in the lower pane displays the following fields for all outages:

    • Item Type
    • Item No
    • Outage Start

    • Outage End

    • Downtime

    • Outage Type

    • Open Status

    • Linked CI

    • Description

    • Creating Analyst

    • Closing Analyst

  6. Update the details as required and save the changes.
