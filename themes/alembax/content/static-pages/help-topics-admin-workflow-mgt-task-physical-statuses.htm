Task Physical Statuses

A task's physical status is the Current and Forward status displayed on the Tasks Outstanding window. You cannot create new status titles, but you can rename them.

Renaming Task Physical Statuses

Before you start

If your system is partitioned, ensure that you are working in the appropriate partition.

You must have Workflow Management Set Up enabled within your General Access Security Role to work with task phases.

  1. Select â‰¡ > Admin > System Administration. The System Administration window appears. In the Explorer pane, expand Workflow Management.
  2. Select the Task Physical Statuses option. The Task Physical Statuses window appears. The Title column displays the system-defined names and the Name column shows Admin assigned names. You can adjust the column widths if required.
  3. In the Name column, select the name you want to change and type the new name you want to use.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.
