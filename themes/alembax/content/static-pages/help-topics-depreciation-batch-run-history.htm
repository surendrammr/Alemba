Batch Run History

This part of the online help is currently in development.

You can viewing scheduled batch runs and the depreciation calculations performed against your CMDB using the Batch Run History option within Depreciation on the System Administration Explorer pane.

Viewing a List of Batch Runs

Before You Start

Enable the Depreciation Setup within your General Access Security Role in order to configure any Depreciation administration settings.

You must first configure your Depreciation Schedule and Depreciation Methods before configuring batch runs.

You must have system administrator privileges to configure batch runs.

  1. Select ≡ > Admin > System Administration.

    The System Administration window is displayed.

    In the Explorer pane, locate the Depreciation group. You may need to expand the group to see the options within it.

  2. Select Batch Run History to display the window.
  1. The window displays all scheduled batch runs and the depreciation calculations performed against your CMDB. For each depreciation batch run that has occurred, this table displays the following information:

    2. Batch Run Number This is a unique reference number for the batch run. This number is allocated by ASM Core for each batch run
    Depreciation Schedule Name The depreciation schedule selected for the batch run
    Month Ending The month and year the depreciation calculation was performed up to
    Execute Start Time the batch run began
    Execute End Time the batch run finished
    Analyst The name of the Analyst who defined the batch run
    Filter Options (CMDB Type, CMDB, User, Organization, Location) Filter options that were selected for the batch run
  2. These columns are the default settings. You can change the columns you wish to display.

    3. You are unable to edit the fields as they are only for reference purposes.

Viewing Details of Selected Batch Run

  1. Select ≡ > Admin > System Administration.

    The System Administration window is displayed.

    In the Explorer pane, locate the Depreciation group. You may need to expand the group to see the options within it.

  2. Select Batch Run History to display the window.
  1. Select the batch for which you want to see more details by double-clicking on the row or highlight the batch row and then select .
  2. The Depreciation Write Down for Selected Batch window will now be displayed. This shows non-editable details of the batch you have selected. These fields are for reference purposes only. The current batch run number is shown in the Batch Number field. The information for the depreciation calculation is shown in the table. You can adjust the column display if you wish.
