Change Reasons

Changes to CMDB items require a change reason if the Change Reason option is selected in CMDB Settings.

When you save a change, the Change Reasons box will be displayed .

Complete the details and select OK to record the change reason and save the changes.

The reasons given appear in the Administration Audit Trail (for configuration settings) and on any screen, form or message template history for changes made through ASM Designer.
