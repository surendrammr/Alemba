Connecting Alemba Service Manager™ to the Dashboard Server

Before you can connect Alemba Service Manager™ to the Dashboard Server, you must first configure ASM to connect to the User Management Server. For more information, see Connecting Alemba Service Manager™ to the Dashboard Server.

To connect Alemba Service Manager™ to the Dashboard Server:

  1. In the System Administration Explorer pane, expand Dashboards, then select Dashboard Server Settings.

    The Dashboard Server API Connection Details window appears.

  2. In Application URL, type the Dashboard Server URL in the following format: http://ServerName/AlembaUMS/, where ServerName is the server name or IP address.

    For example, http://localhost/AlembaDashboardServer

  3. In User ID, type the Dashboard system administrator you set up.

    For example, AlembaAdmin.

    For more information, see Configuring the Dashboard Server.

  4. In User Password, type the Dashboard system administrator password.

  5. In Validation Key, paste the Dashboard Server web.config validation key.

    For more information, see Accessing the Validation and Decryption Keys.

  6. In Decryption Key, paste the Dashboard Server web.config decryption key.

    For more information, see Accessing the Validation and Decryption Keys.

  7. Select to save your changes.
  8. Select to test the connection.
