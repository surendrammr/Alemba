IPK Management Security Role: IPK Statuses Tab

Before you start

You must have Security Roles setup selected in the Admin tab of your own General Access security role before you can assign or remove permissions for any security roles.

This tab is only shown if you have Enable IPK Statuses and Streams selected in the IPK Settings . Otherwise you will see the Screen Sets tab.

IPK statuses must be defined before you can give access to them via this tab.

You can specify which IPK statuses Analysts with the security role can assign to a call. You can also set a default IPK Status and IPK Stream for the role, so that an Analyst with this role can simply click the Call button on the menu to launch the Call Details window linked to the default status and stream. To enable this, the Default IPK Stream/Status option in the Options tab must also be selected for this role.

You must ensure that the role has access to streams and IPK Statuses that are linked. If not, Analysts with this role will not be able to log a call.

  1. Display the IPK Management Security Roles details window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Security Roles, then select IPK Management. All existing IPK Management security roles are displayed.

    The pre-defined role All typically has all role options selected. However, some of the options are not selected by default and you should explore the permissions set before assigning the role.

    If you want to create a role, select and assign the role a name in the Role Name field. To view or update a role, select the role in the browse table and select .

  2. Select the IPK Statuses tab . This tab displays two tables. The upper one shows a list of all available IPK statuses. The lower one shows the IPK statuses that are linked to the current role.
  3. Select IPK Status Security to give Analysts with this role access to particular IPK statuses. Deselect the option to give the role access to all IPK statuses.
  4. To add a specific IPK status to a role, select the status in the Select IPK table and then select .
  5. To remove an IPK status from a role, select the status in the Selected IPK table and click Remove.
  6. To select an IPK status as the default for the role, select it in the Selected IPK table and click Set as Default. If you select an IPK Status that is linked to the default IPK Stream for this role, the IPK Status is set as the default. If the IPK Status is not linked to the current default IPK Stream, a warning message appears. Select Yes to select the IPK Status as the default and update the default IPK Stream to the topmost IPK stream linked to the status. If the IPK Status is not linked to any IPK Streams, the default IPK Stream is set as the default.

    7. Once you specify a default IPK status, you cannot leave this field blank, you can only select from the available statuses.

  7. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so. Alternatively, select another tab, if appropriate.
