Organization Links Explorer Option

Organizations can be included in the CMDB as “sites” or “departments” and linked to a Parent Organization. The relationship between the current organization and other organizations appears in the Organization Link window.

  1. Display the Organization Details window if it is not already visible on screen, by searching for the organization.
  2. Select to view the details in edit mode if necessary.
  3. Select the Organization Link option from the explorer pane. This appears within the Links group. Locate the group, and expand it if necessary.
  4. The Organization Link window appears, showing the entire hierarchy of the current organization; this includes any parent organizations to which it is linked, as well as any child organizations linked to the one currently selected. The organizations linked to any of the parents or children of the current one are also displayed. From here, you can do any of the following, using the buttons on the toolbar:

    5. To return to the <Entity> Details window
    To upload a file
    To upload an image
    To download a file
       

    To delete an attachment

    You must have Object Delete selected in your Bulletin Board security role to access this option.
  5. From here, you can select any of the following:

    6. Select To...
    copy the link diagram
  6. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.
