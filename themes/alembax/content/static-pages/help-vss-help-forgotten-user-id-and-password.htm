Forgotten User ID and Password (Self Service Portal)

If you have forgotten your Login ID or password, you can submit a request to retrieve them.

  1. From Self Service Portal Login page, select the link to have your User ID and Password sent to you via email. If this is not visible, select Forgot Password in the top left corner of the window.
  2. The Forgotten Password page appears.
  3. In the Email field, type the email address you provided when you first registered for using Self Service Portal.
  4. Click OK. A confirmation appears. Your Login ID and Password are sent to the email address you entered. You can use these details to log in to Self Service Portal.

Troubleshooting

Your email address was not found. Ensure you have entered it correctly.

You may not be an existing registered user or you have typed an email address that was different from the one that was used during registration.

If you are unsure of the email address you typed during registration, contact your System Administrator.

If you have not registered, select the link to register.
