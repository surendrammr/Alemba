Budgets Explorer Option

You can specify the estimated time and cost associated with the current task, as well as to view the actual time and costs involved with the task. This is particularly useful to monitor the progress of your tasks and requests compared with your estimates.

  1. Select the Budgets option in the Task Details explorer pane to display the Task Budget window.
  2. This window displays two sections:

    3. Budget the estimated (budgeted) time (under the heading ) and costs for the task
    Actual the actual time and cost for the task. As action is taken on the task (and expenses are incurred), ASM Core calculates all costs and expenses on the task and displays them in the corresponding fields in this section. This enables you to compare the actual time, costs and expenses with those estimated.
  3. You can only enter the budgeted time and costs for this task on the window. Complete the details:
  4. Time an estimate of time spent actioning the task. This time must be expressed in hours and minutes (such as 40)
    Cost specify any internal costs involved
    Expenses specify any external costs involved (such as the cost of materials, service contracts and so on)

As the task is actioned and expenses are incurred, the system tallies these up and displays them as the Actual values. These can be compared to the Budget values, providing an idea of how well the request and task(s) are progressing.

This can be viewed on the Request Details window under the Budgets tab.
