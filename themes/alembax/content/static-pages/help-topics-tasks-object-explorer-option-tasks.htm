Object Explorer Option - Tasks

You can attach files from Windows applications to the current task. These files can, for example, provide additional information about the task to contribute to its completion.

Before you start

Your ability to view, update, cancel and delete attached files is set in the Tasks tab of your Workflow Management Security Role settings.

  1. Select the Object explorer option on the Task Details explorer pane to view the Attach Object to Task window. Using this window, you can attach objects or files from Windows applications to the current task.
  2. The Attach Object to Task window displays all the objects that have been attached to this task, and to the request to which this task belongs. You can perform all object management tasks on this window.
  3. This window provides the following Upload/Download filter options:

    4. Task Objects Select this option to view all the objects linked to the current task
    Request Objects

    Select this option to view the objects attached to the current request and to all tasks that are a part of this request. In this mode, you can use to link objects linked to the request or other tasks to the current task. When you select the Upload/Download Task Objects option, all linked objects will also be displayed.
