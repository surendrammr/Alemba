Contracts Explorer Option

The Contracts explorer option enables you to link external supplier contracts with the Analyst. This indicates that the Analyst is the point of contact for the outside company.

Linking a Contract to an Analyst

Before you start

The person must be an Analyst.

  1. From the Person Details Explorer pane, expand the Data tab.
  2. Select Contracts from the menu options. The Person Linked Contracts window appears, displaying a browse table containing the person's existing contracts and organizations with which they are held. Select Include Removed to view the deleted contracts.
  3. Select to make the buttons beneath the table active if necessary.
  4. Select Add.
  5. When the Contract Search window appears, specify the search criteria and then select Search to display a list of contracts in the browse table. Then select the contract you want to link to the person and select .
  6. The Person Linked Contracts window appears once more, with the selected contract displayed in the browse table.
  7. Add contracts for which the analyst is responsible.

Unlinking a Contract from an Analyst

  1. From the Person Details Explorer pane, expand the Data tab.
  2. Select Contracts from the menu options. The Person Linked Contracts window appears, displaying a browse table containing the person's existing contracts and organizations with which they are held. Select Include Removed to view the deleted contracts.
  3. Select to make the buttons beneath the table active if necessary.
  4. Select the contract you want to unlink, and then select Delete.
  5. Confirm the deletion. The contract disappears from the browse table.

  6. To display the deleted contracts, select the Include Removed checkbox.
