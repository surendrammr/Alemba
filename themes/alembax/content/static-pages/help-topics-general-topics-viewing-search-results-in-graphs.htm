Viewing Search Results in Graphs

You can view calls, requests, and tasks in a graphical format from their corresponding Search window.

  1. Search on the entity.
  2. When the results are displayed in a browse table on the Search results window, select the button.
  3. The results are displayed in a bar chart and the following options are available:
    4. Select To...
    Display the results in a table format
    Choose between different graph formats , eg Bar, Line, Pie, and settings.

    Use Group By to select the search criterion on which the graph is based.

    Maximum Points determines the total number of horizontal bars displayed on the window. If you are displaying a pie chart, only slices that have values greater than the Minimum Point Value will be shown. Lower values will be grouped into an “Other” slice.
    Print the graph
    Copy the graph
  4. Select from the options or double-click any section of the graph to open a new search window with search criteria based on the chosen section.
