Searching for CMDB Items

This topic relates to the Nano interface.

You can search for CMDB Items in the system using the CMDB Search window.

On the CMDB Search window you can:

  • Use the Standard Search to find CMDB Items by name, ref, or organization.
  • Use the Advanced Search to find CMDB Items using all CMDB Item search fields.
  • Configure the columns by adding or removing columns and changing their order.
  • Right-click on a CMDB Item record to open the CMDB Item Details window or save the details to PDF.

Using the CMDB Search Window

Before you start

To view service details, CMDB Item View must be selected in the CMDB Items tab of your Configuration Management Security Role.

To edit service details, CMDB Item Update must be selected in the CMDB Items tab of your Configuration Management Security Role.

  1. On the menu, select Search > CMDB Item Search.

    The CMDB Search window opens with the Standard Search criteria fields displayed by default.

    2. More advanced search options are available in the Advanced Search.

  2. Enter criteria for your search in the Standard Search fields:
Resource Title

Type part or all of the title of the CMDB Item. The search automatically runs when your cursor leaves the field.
Resource Ref

Type part or all of the ref of the CMDB Item. The search automatically runs when your cursor leaves the field.
Resource Type From the list, select the CMDB Item Type.
Organization Type part or all of the name of the CMDB Item's organization. Matching results are displayed beneath the field. Select one from the list.
  1. Records matching your search criteria are displayed in the Results table.
  • Sort the table of results by clicking on a column header.
  • Change the columns in the table by selecting from the toolbar to open Column Configuration window.

    • See Configuring Columns for more details.

  1. To open the CMDB Item details, select the CMDB record(s) in the results table and double-click, or right-click and then select .

    2. You can select multiple CMDB Items by holding the CTRL key while select each row.

  2. Other actions are available by selecting the record(s) in the results table, then right-clicking and selecting:
  • to open the CMDB Linking Diagram.
  • to generate a Details Report.

Advanced Search

Advanced search criteria options are available if the Standard Search does not return the records you are looking for.

  1. Switch to Advanced Search by selecting from the toolbar. The Advanced Search panel opens on the right of the search window.
  2. To minimize the Advanced Search window and retain the search criteria, press . To expand it again press on the toolbar.
  3. To close the Advanced Search window, clear the criteria, and return to the Standard search, select on the toolbar.
  4. Enter the criteria for your search and then press .

Advanced Search criteria (default) options:

Configuration Items Searches for Configuration Items
Services Searches for Services
Service Actions Searches for Service Actions
Service Bundles Searches for Service Bundles
Active Searches CMDB Items that are not deleted
Deleted Searches CMDB Items that are deleted
Template Includes template CMDB Items in the search
Partition

This option is only visible if you have access to more than one partition.

Use the drop-down list to select from the partitions you can access.
Include Deleted Field Values Selecting this option enables deleted values for list fields to be selected.

If you are searching for a CMDB Item by the Criticality status and select the Include Deleted Field Values option; the Criticality list will be updated to display names of statuses that you created and subsequently deleted. You can then select a deleted value from the list, and conduct your search.
Ref Type the Ref value of the CMDB Item.
CMDB Item Type

From the list, select the Type of the CMDB Item you are searching for. Depending on the number of tiers configured for your system, you may see one to five tiers. You can select a type from any level.

  • To search for CMDB Items that match only the selected type, select the Equal option.
  • To search for CMDB Items that match the selected type and the tiers below, select Equal + Below.
  • To search for CMDB Items that match the selected type and above, select Equal + Above.
User Type part or all of the user's name. Matching results are displayed beneath the field. Select one from the list.
Organization

Type part or all of the organization's name. Matching results are displayed beneath the field. Select one from the list. Organizations can be linked in a hierarchy.

  • To search for calls that match only the selected organization, select the Equal option.
  • To search for calls that match the selected organization and those linked as children, select Equal + Below.
  • To search for calls that match the selected organization and those linked as parent, select Equal + Above.
Location Type part or all of the organization's name. Matching results are displayed beneath the field. Select one from the list.
