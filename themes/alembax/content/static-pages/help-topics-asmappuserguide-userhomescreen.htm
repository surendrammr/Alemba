The ASM app Home Screen (User login)

The following home screen appears when you log in as a User. It provides a quick overview of items that are currently active in the system. Your access is based on your permissions for Self Service Portal.

The tiles display the following information.

Calls

The number of open calls that have been created by or for you

Approvals

The number of approvals that have been forwarded to you

Orders

The number of orders that you have submitted that are currently being processed

Assets

The number of assets for which you are recorded as the user

You can only view those calls, approvals, orders and assets that are visible to you as a user in the Self Service Portal.


You can access further options by selecting the menu button. This varies depending on the phone type.

iOS Tap on the button in the bottom left hand corner
Windows Tap on the button in the bottom right hand corner
Android Tap on the button in the top right corner
