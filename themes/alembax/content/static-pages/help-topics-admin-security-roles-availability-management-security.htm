Availability Management Security Roles

You can control permissions for viewing, creating and updating Outages, Schedules and viewing Availability Data for CMDB items using the Availability Management security roles.

Before you start

You must have Security Roles setup selected in the Admin tab of your own General Access security role before you can assign or remove permissions for any security roles.

  1. Display the Availability Management Security Roles details window, if it is not already on screen.

    Select â‰¡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Security Roles, then select the type of role you want to view.

    A browse table appears, displaying all of the existing roles of this type. You can create, update or rename a security role of this type.

    The pre-defined role All typically has all role options selected. However, some of the options are not selected by default and you should explore the permissions set before assigning the role.

    If you want to create a role, select and assign the role a name in the Role Name field. To view or update a role, select the role in the browse table and select .

  2. Assign the permissions you want for the security role :
Create Outage From Call

Analysts with this role can create an outage record while logging an incident
Create Outage From Task

Analysts with this role can create an outage record from a task
Create Outage From Request

Analysts with this role can create an outage record from a request
Outage Update

Analysts with this role can update the details of an outage
Browse Outages

Analysts with this role can browse through a list of outages and view their details
Create Schedules

Analysts with this role can create availability schedules
View Availability Data

Analysts with this role can view the availability information for CMDB items.

Selecting this option enables the Availability option on the CMDB Item Explorer
Set Schedules and Targets

Analysts with this role can set the availability for CMDB items, and set targets against the availability
Set Public Holidays

Analysts with this role can designate a defined period as public holiday/s in the availability schedule
  1. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so.
