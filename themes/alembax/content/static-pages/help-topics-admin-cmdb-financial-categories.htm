Financial Category

You can define financial categories which can be used to add financial information about CMDB items. Analysts can select a financial category when specifying the CMDB item details.

You may define as many financial categories as you wish. You can create, rename and delete these financial categories.

Creating a Financial Category

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Financial Category to open the window .
  1. Select . A new row is added to the browse table.
  2. In the Name list, type the name or description of the new financial category.
  3. Repeat for each financial category you want to create.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

    5. Once you select , the browse table displays the entries in alphabetical order.

Renaming a Financial Category

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Financial Category to open the window .
  1. Select the financial category entry you want to edit.
  2. Type the name you want to assign for the selected financial category.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.

    4. Once you select , the browse table displays the entries in alphabetical order.

Deleting a Financial Category

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

You cannot reverse this procedure. If you delete a financial category by mistake, you have to re-create the entry by creating a new financial category.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Financial Category to open the window .
  1. Select the financial category entry you want to delete.
  2. Select . If you attempt to delete the system default, you will receive a warning message.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
