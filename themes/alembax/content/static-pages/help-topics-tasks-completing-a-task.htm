Completing a Task

When you have finished taking action on a task, you can complete the task.

  1. Open the task from either the Tasks Outstanding or Request Details window. The Task Details window appears.
  2. Expand the Action Information section and record the action you have taken on the task, including comments, times and expenses.
  3. Click Complete, or select the Complete explorer option.

The task is no longer displayed on your Tasks Outstanding window and its status updates on the request managerâ€™s Request Details window.

All outstanding tasks are closed when the request is completed, regardless of whether the task is complete.
