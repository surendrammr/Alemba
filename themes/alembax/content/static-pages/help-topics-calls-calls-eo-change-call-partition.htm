Change Call Partition Explorer Option

The Change Call Partition explorer option enables you to change the partition of the current call to any partition to which you have access.

This explorer option is only enabled if:

  • Your system is partitioned
  • Change Partition on Take Action is selected in the Partition Settings window,
  • Change Call Partition is selected in the Options tab of your IPK Management Security Role
  • You have been assigned access to several partitions
  1. Display the Call Details window if it is not already on screen.
  2. If you are in review mode, click the Take Action button.
  3. In the Explorer pane, expand Workflow, then select Change Call Partition. The Select Call Partition window appears with a list of partitions to which you have access.
  4. Select the partition to which you want to allocate the call.
  5. Click OK.

The call partition is updated and displayed in the Information panel. An entry is also added to the call history to record the change.
