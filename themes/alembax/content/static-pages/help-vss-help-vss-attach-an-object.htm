Attaching Objects

Objects can be added to open incidents, requests and approvals, as well as config items. Objects can be documents or files that add more information.

This feature is only available on SQLÂ systems.

  1. Display the Incident, Request, Approval or Config Item details.
  2. Select the Add Attachment button at the top of the page.
  3. The Add an Attachment to your <Entity> page appears. Complete the details.

    4. <entity> number The number of the config item, incident, request or approval to which you are adding the object(s). This is prefilled, based on the previous screen.
    Attach File Select the Browse button and navigate to the location of your file(s). Select one or more files from this location. You can select multiple files from multiple locations.
    Description Type a brief description of the details of the attachment(s).
    Send Email Select to send an email to the analyst to whom the incident/request/approval is assigned.
    Send Pager Select to send a pager to the analyst to whom the incident/request/approval is assigned.
  4. Click OK. An acknowledgment appears.

After attaching an object, if you reload the page, you will need to attach the object again. Uploaded objects are cleared when a page refreshes to display an error message in case a mandatory field has not been completed on that page.
