Closing Requests

This topic relates to the Nano interface.

Nano provides multiple methods for closing a request:

  • From the Request Details window
  • From the Request Search and Request Outstanding windows

When closing requests from the Search or Outstanding windows you must complete the Request Close window. Also, you can select and close multiple requests in one action.

Before you start

To view requests Workflow Analyst must be selected in the Shared tab of your Workflow Management Security Role.
