Task Types

You can record the task type on each task, such as Installation, or Testing. They indicate the nature of the task being logged.

Analysts select task types from the Task Type field on the Task Details window.

Creating a Task Type

Before you start

You must have Workflow Management Set Up enabled within your General Access Security Role to work with task types.

  1. Select ≡ > Admin > System Administration. The System Administration window appears. In the Explorer pane, expand Workflow Management.
  2. Select the Task Types option. The Task Types window appears, with a browse table of existing task types .
  3. Select . A blank row is inserted in the browse table.
  4. Type the name of the task type.
  5. Repeat for each task type you want to create.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so.

Renaming a Task Type

Before you start

You must have Workflow Management Set Up enabled within your General Access Security Role to work with task types.

Do not rename the system-defined “Unspecified” task type.

  1. Select ≡ > Admin > System Administration. The System Administration window appears. In the Explorer pane, expand Workflow Management.
  2. Select the Task Types option. The Task Types window appears, with a browse table of existing task types .
  3. Select the task type you want to rename, and overtype with the new name.

    4. Do not rename the system-defined “Unspecified” task type.

  4. Repeat for each task type you want to rename.
  5. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a Task Type

Before you start

You must have Workflow Management Set Up enabled within your General Access Security Role to work with task types.

You cannot delete the default task type (Unspecified). You cannot restore deleted task types.

  1. Select ≡ > Admin > System Administration. The System Administration window appears. In the Explorer pane, expand Workflow Management.
  2. Select the Task Types option. The Task Types window appears, with a browse table of existing task types .
  3. Select the task type you want to delete. Select . If you attempt to delete the system default, you will receive a warning message.
  4. Repeat for each task type you want to delete.
  5. Select to save the changes. Provide the Change Reasons if prompted to do so.
