Working with Resources

This topic relates to the Nano interface.

Before you start

Read Introduction to Nano and the Nano Feature List before continuing.

Nano provides a range of methods for working with the resource types available in Nano, to suit different working styles.

This topic provides an overview of the methods available to perform actions on knowledge articles, person records, CMDB Items, Organizations, and Locations.

For detailed information on working with each of these resources, use these links:

  • Using the Knowledge Details Window - describes use and function of fields on the Knowledge Article details window
  • Searching for Knowledge Articles - explains how to search for Knowledge Articles Items
  • Using the CMDB Item Details Window - describes use and function of fields on the CMDB Item details window
  • Searching for CMDB Items - explains how to search for and view CMDB Items
  • Using the Person Details Window - describes use and function of fields on the Person details window
  • Searching for Person Records - explains how to search for and view Person records
  • Using the Organization Details Window - describes use and function of fields on the Organization details window
  • Using the Location Details Window - describes use and function of fields on the Location details window

Working with Knowledge Articles

Search for knowledge
  1. On the menu select Search, then Knowledge Search
  2. Complete the Knowledge Search fields.
  3. Knowledge articles matching your parameters are displayed
View knowledge details
  1. From Knowledge Search, find the knowledge article and use any of these methods:
  • Double-click the knowledge article
  • Right-click the knowledge article and select
  1. The Knowledge Details window appears.
Update knowledge
  1. Open the Knowledge Details
  2. select Edit to make the fields editable
  3. Update the Knowledge Details fields
  4. Then select Save. The Knowledge Save window appears
  5. Complete the Knowledge Save fields
  6. select OK.

Working with CMDB Items

Search for a CMDB Item
  1. On the menu select Search, then CMDB Item Search
  2. Complete the CMDB Item Search fields.
  3. CMDB Items matching your parameters are displayed
View CMDB Item details

Method 1: CMDB Item Search

  1. Search the table of results for the CMDB Item you want to view
  2. Open the record using any of these methods:
  • Double-click on the record; or
  • Right-click the record and select
  1. The CMDB Details window appears.

Method 2: Details of a call, request, or task

  1. On the details screen of the call, request, or task, find the CMDB Item field you want to view
  2. To see the CMDB Item Details window, double-click on the CMDB Item name in the field
  3. To see the CMDB Item Review window, click once on the CMDB Item name in the field
  • From here you can select to open the CMDB Item Details window
Update a CMDB Item
  1. Open the CMDB Item Details window
  2. select Edit if the details window is in review mode
  3. Update the CMDB Details fields
  4. select Save.

Working with Person Records

Search for a Person
  1. On the menu select Search, then Person Search
  2. Complete the Person Search fields.
  3. Person records matching your parameters are displayed
View Person details

Method 1: Person Search

  1. Search the table of results for the Person record you want to view
  2. Open the record using any of these methods:
  • Double-click on the record; or
  • Right-click the record and select
  1. The Person Details window appears.

Method 2: Details of a call, request, or task or other entity with a Person field:

  1. On the details screen of the call, request, or task, find the Person field you want to view
  2. To see the Person Details window, double-click on the Person's name in the field
  3. To see the Person Review window, click once on the Person's name in the field
  • From here you can select to open the Person Details window
Update a Person
  1. Open the Person Details
  2. select Edit if the details window is in review mode
  3. Update the Person Details fields
  4. select Save.
