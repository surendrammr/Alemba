Chat in Self Service Portal

You can log calls in Self Service Portal via a text-based chat with a member of the Service Desk support team.

You must have Chat User selected in the Options tab of their Self Service Portal role.

  1. Once you have chat user permissions, you can see the Chat option in your Self Service My Options menu. If there is an analyst available, the item will appear as simply Chat - . If there are no analysts available, either because they are offline or dealing with other users, the option title will reflect the status - .
  2. When you select the option, the Chat Request window appears. (If it selected when there are no analysts available, the Description field is grayed out.)
  3. You complete the description and select Chat Request.
  4. This logs a call, which appears on the screen, and opens a Chat window showing the status of the chat, the number of available analysts and your place in any queue. This is updated as the status changes.
  5. When the chat session is accepted by an analyst, the chat begins. Users and analysts can typeto a chat box at the bottom of the chat window, sending it by either pressing Enter or selecting Send.
  6. The icons at the top of the chat window indicate the status of the chat. Typically, there will be a green dot, a minimize and a close icon. When the analyst is typing, the green dot icon will display a small speech bubble - .
  7. You can end the chat by closing the window. The details of the call logged as a result of the chat appears on the screen, and when you select the Chat option again, you can see the call in the list of Recently Logged Calls.
