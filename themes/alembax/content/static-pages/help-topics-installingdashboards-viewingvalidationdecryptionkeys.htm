Accessing the Validation and Decryption Keys

In order to complete the connection between Alemba Service Managerâ„¢ and the Dashboard Server, you need to copy the Dashboard Server Validation and Decryption Keys from the web.config file and paste them into the ASM Dashboard Settings.

To view the Validation and Decryption Keys in the web.config file:

  1. Open Web.config using Notepad.

    By default, this file is in your c:\Syncfusion\DashboardServer.Web folder, where c is the

    Windows drive where you installed the Dashboard platform.

  2. Search for validationKey, then select the key:

  3. Right-click the selection, then select Copy on the shortcut menu.
  4. Paste the selection into the ASM Dashboard Settings Validation Key field.

  5. Search for decryptionKey, then select the key:

  6. Right-click the selection, then select Copy on the shortcut menu.
  7. Paste the selection into the ASM Dashboard Settings Decryption Key field.
