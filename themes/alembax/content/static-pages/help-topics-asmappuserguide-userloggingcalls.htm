Logging Calls in the ASM app (User login)

You can log calls as a user using the ASM app.

Before you start

You must have permission to log a call in the IPK Management Role: Options Tab of your IPK Management security role.

See the Prerequisites checklist for supported operating systems.

  1. Logging in to the ASM app as a User.
  2. From the home screen, select the menu button.
    iOS Tap on the button in the bottom left hand corner
    Windows Tap on the button in the bottom right hand corner
    Android Tap on the button in the top right corner

  3. Select the Log Call option.

  4. If you are using an iOS phone, you will see a screen in which you can select the call template.


    5. Call templates are used to make logging calls easier. Different templates are used for different call or problem types, as they often require different information from the user.

  5. Select the template you require, and then select Done.

    6. You can only see call templates that you have permission to use to log the call.

    Although the call templates used are taken from ASM Core, the call template does not pre-populate fields with information in the app, as it does in ASM Core.

  6. The Log Call screen appears.


  7. Complete the details.

    8. Template Select the Call Template you want to use. This is a mandatory field. The Call template is used to determine what fields are required/displayed.

    This field is only displayed if there are several templates to choose from. It is not displayed on iOS phones, as this information has already been supplied. You only have access to permitted templates.

    Although the call templates used are taken from ASM Core, the call template does not pre-populate the fields with information in the app, as it does in ASM Core.
    Service Select the service you wish to log the call against. You will only see the services that you have permission to access.
    Asset Specify the asset you wish to log the call against. You can choose from a list of configuration items. You will only see the items that you have permission to access.

    This list does not include services. Assets in the app equate to Resources in Self Service Portal.
    Description Type a description of the issue. This field is mandatory.
  8. When you have completed the details, select LOG.

When a call is logged via the app, the logging method will be declared as 'Portal'.
