Accessing the User Management Server client ID

You can find the User Management Server client ID in the Dashboard Server User Directory Settings.

  1. Using Dashboard Server, select to view Settings.
  2. Select the User Directory tab.
  3. If required, highlight the Client Id, then right-click and select Copy on the shortcut menu.
