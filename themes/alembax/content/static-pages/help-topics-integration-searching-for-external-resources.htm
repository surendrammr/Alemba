Searching for External Resources

The External Resource Search window enables analysts to search on the following criteria:

Connector This list enables you to select the third party application on which you want to locate resources
Source This list displays the sources configured for the selected connector
Resource Type This list enables you to select a resource type when the connector and source are selected. The values listed here correspond to the connector and source you have selected.
Resource Type the name of the external resource on the relevant source. This will retrieve external resources by their alias.
Imported Resources Only

This is selected by default. While selected, a Cached Resource Search is performed, and the search returns both active and inactive imported resources.

You must have Federated CMDB Admin permissions to deselect this option.
Partition If you are using partitioning on your system, select the partition in which the external resource is located. This list displays all the partitions to which you have access.

If External Resource criteria are not specified, the search returns a list of matching sources. Otherwise, a list of matching primary resources across the relevant sources is returned. A message appears if the system is unable to retrieve any matching external resources. The toolbar displays the alias of the currently selected source or resource. If neither is selected, the caption is not displayed.

Search Modes

The External Resource Search window can be in one of three modes, depending on the window from which you access it:

General Search

The window is in General Search mode if the search was launched from the Search dropdown menu.

CMDB Item Search

Type the item name to automatically search the database.

Resource Search

The window is in Resource Search mode if the search was launched from the CMDB Item Linked Resources window. In this mode, the toolbar displays the Select Resource button. When a resource has been selected in the External Resource Search results, this button becomes enabled. Clicking this button links the selected resource to the CMDB Item from which the search was run.
