Closing Tasks

This topic relates to the Nano interface.

Nano provides multiple methods for closing a task:

  • From the Task Details window
  • From the Task Search window and Task Outstanding windows

When closing tasks from the Search or Outstanding windows you must complete the Task Close. Also, you can select and close multiple tasks in one action.

Before you start

To view tasks Workflow Analyst must be selected in the Shared tab of your Workflow Management Security Role.
