Cost Center Category

You can define cost center categories such as company departments or geographical locations. Analysts can then select a cost center category when adding Cost Centers.

You may define as many cost center categories as you wish.

Creating a Cost Center Category

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select the Contract Center Category option from the Explorer pane. The Cost Center Category window appears, allowing you to define categories when adding cost centers.
  1. Select . A new row is created to the browse table.
  2. In the Name list, type the name of the new cost center category you want to create.
  3. Repeat these steps for each cost center category you want to create.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

The List of Cost Center Categories displays alphabetically once saved.

Renaming a Cost Center Category

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select the Contract Center Category option from the Explorer pane. The Cost Center Category window appears, allowing you to define categories when adding cost centers.
  1. Select the cost center category you want to edit by selecting on it. The entry is highlighted.
  2. Type the name you want to assign for the selected cost center category.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a Cost Center Category

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

You cannot reverse this procedure. If you delete a cost center category by mistake, you have to re-create the entry by creating a new cost center category.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select the Contract Center Category option from the Explorer pane. The Cost Center Category window appears, allowing you to define categories when adding cost centers.
  1. Select the cost center category you want to delete from the list.
  2. Select . If you attempt to delete the system default, you will receive a warning message.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
