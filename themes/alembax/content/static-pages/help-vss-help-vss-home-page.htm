Self Service Portal Home Page

The Self Service Portal home page is highly configurable, so your screen may look different to the one illustrated below, but the key elements will remain the same.

There are two Home Page layouts - default and classic.

  • The default layout contains two columns of widgets, displaying summary information about your workload and latest news. If there are more entries in the widgets than can be displayed, you will see pagination options at the bottom of the widget, and you can scroll through the pages to view the entries.
  • The classic layout has a single column of widgets, which expand and contract to accommodate the entries within it.

The layout for your Self Service Portal display is chosen by your system administrator in the Self Service Portal Systems window.

My Options

The My Options pane is down the left hand side of the window and displays all of the menu options that your system administrator has made available. The menu names and color combinations have also been configured by your system administrator. You may small arrows to the right of some of your menu options. This indicates that this item is, in fact, a menu of options in itself.

Depending on how your system administrator has configured your system in the Self Service Portal My Options window, if you hover over these items, additional menu options will appear in a pop out submenu, or you can click on them to expand and contract the additional menu options.

Do not use the Back and Forward buttons on your web browser when in Self Service Portal. You should always use the Back button provided on the actual Self Service Portal page is you want to navigate to the previous page, or use the navigation bar.

Widgets on the Home Page

If your system administrator has configured Self Service Portal to have the default home page layout, you will see two columns of widgets containing summary information. If the contents exceed the display area, you can use the pagination options at the bottom of the widget to scroll through the entries. You can hover over any entry to highlight it, and then click to display the details.

System administrators configure the columns that are displayed, the order in which they are displayed, and the manner in which they are sorted. See Self Service Portal Tables for more details.

The most recent Self Service Portal Bulletin Board messages are displayed under Latest News.

My Counts

This widget shows a count of currently active incidents, requests, approvals that have been forwarded to you, as well as configuration items for which you are a linked user.

Latest News

This widget shows the most recent messages on the Bulletin Board. Click the More News link to see the News Feed page, displaying more entries. You can drill down on each of the entries by clicking on them. Messages may contain file attachments and URL links. File attachments can be downloaded and saved to your local computer, and you can select the URL links.

Major Incidents

This widget is only visible to users with permissions to review major incidents.

In this widget, you can see all of the current major incidents.

If you have the necessary permissions, you can add yourself to a Major Incident so that the its full impact can be captured and reported on, and you can receive feedback and updates, such as when the Major Incident is resolved. You will see an Add Me box in the Major Incidents widget. Once you choose to be added to the major incident, an incident will be cloned and assigned to you and the Add Me button will disappear. You can then review the major incident, add notes and attachments as standard.

You can also select the major incident to view the details, at which point you can choose to add a note, add an attachment, or Add Me.

My Incidents

This widget displays all open incidents where you are the linked user.

The incidents are displayed in chronological order by the call number. To find other incidents, select the Search Incidents link. The Review Past/Current Incidents window appears.

Most columns in this widget are self explanatory.

The Indicator column, if displayed, contains icons, which can be interpreted as follows:

Indicator Description

A note has been added to the call.

The call has other calls linked to it as children, that is, this call is a parent call.

The call is linked to another call as a child, that is, this call has a parent call linked to it.

The CI in the call has an outage.
The call has parent and child calls linked to it.
The call is logged on behalf of a person flagged as a VIP.

My Requests

This widget displays all open requests where you are the linked user.

The requests are displayed in chronological order by the request number. Any Shopping Cart Orders which have been logged will also display in this table.

To find other requests, select the Search Requests link. The Review Past/Current Requests window appears.

My Orders

This widget displays all of your outstanding orders, in chronological order.

To search for other orders, select the Search Orders link. The Review Past/Current Orders link appears.

My Approvals

This widget displays all open User Approval tasks that have been forwarded to you. The approvals are displayed in chronological order by the approval number.

To search for other approvals, select the Search Approvals link. The Review Past/Current Approvals window appears.

My CIs

This widget displays all Configuration Items where you are the linked user.

To search for other configuration items, select the Search Configuration Items link. The Review Configuration Items window appears.
