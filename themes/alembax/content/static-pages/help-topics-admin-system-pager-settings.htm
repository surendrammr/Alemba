Pager Settings

You can configure options for sending pager or SMS text messages from ASM Core.

ASM Core only supports pager messages sent by email.

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

The Analyst(s)’ cell phone/pager number(s) must be entered into the Pager field on their Person Details.

  1. Select ≡ > Admin > System Administration. The System Administration window appears.
  2. In the Explorer pane, expand System, then select Pager Settings. The Pager Settings window appears, displaying the following fields to configure ASM Core for sending pager messages:

    3. Sending Address Type the email address from which all pager messages are marked to be sent.
    Address Format The destination email address of any pager message

    Include the token <PagerID>. When a pager is sent, it is replaced with the receiving Analyst’s Pager ID and indicates to the SMTP server where the pager message is to be sent. An Analyst’s Pager ID is configured on their Person Details.
    Email Subject The subject line of any messages sent

    Include the token <PagerID>.  When a pager is sent, it is replaced with the receiving Analyst’s Pager ID and indicates to the SMTP server where the pager message is to be sent. An Analyst’s Pager ID is configured on their Person Details.

Configuring Settings for SMS Messages (Texts)

You can configure your ASM System to send SMS text messages.

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

The Analyst(s)’ cell phone number(s) must be entered into the Pager field on their Person Details.

  1. Select ≡ > Admin > System Administration. The System Administration window appears.
  2. Select the Pager Settings option. The Pager Settings window appears, displaying the following fields to configure ASM Core for sending pager messages:

    3. Sending Address The email address from which all pager messages are marked to be sent. This is the address which will be recorded as having sent the message, and will receive any undeliverable pager messages.
    Address Format The SMS email gateway address into this field; for example, <pagerID>@sms_email_gateway_address

    Address Format and/or Email Subject must include the token <pagerID>.
    Email Subject The subject line of any messages sent

    Address Format and/or Email Subject must include the token <pagerID>.
  1. Select to save the changes. Provide the Change Reasons if prompted to do so.
