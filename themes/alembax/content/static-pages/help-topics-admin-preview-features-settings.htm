Settings

Preview Features provide early access to upcoming new features. Preview features are still in development and should only be enabled in Test environments.

Enabling Preview Features

Each feature is disabled by default and must be enabled in the Settings window before it can be configured or used.

Some features require additional configuration after being enabled.

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

    In the Explorer pane, expand Preview Features.

  2. Select Settings to display the Preview Features Settings window.
  3. Select each feature you want to enable
    4. Multi Language

    Enables Users to select their preferred language, and see the Self Service Portal in that language.

    English is the default language. System Administrators can import language packs to add more languages.

    Further configuration is required, see Multi Language
    Global Search

    Displays a global search bar in the Self Service Portal, allowing Users to easily search across Knowledge, the Service Request Catalog, calls, and requests.

    For more information, see Global Search
    Call Activities

    Enables the creation and use and of activity items for calls, much like an activity checklist. Analysts can close activity items as they complete them, which ensures the correct process is followed, and enables tracking of call progress. Administrators can create and reuse lists across multiple calls and call templates.

    Further configuration is required, see Call Activities
    Quick Forms

    Allows Administrators to create of lists of pre-defined questions which appear in Call Details screen, improving call resolution time, ensuring important information is captured, and supporting new staff as they get up to speed.

    Further configuration is required, see Quick Forms
    Nano Matching Panel

    Displays buttons on the call details toolbar in Nano that open the Matching Knowledge panel and the Matching Calls panel respectively. These panels present a list of knowledge articles and calls similar to the current call, that may assist in call resolution. Calls and Knowledge articles are matched on fields in the Call details: Type, Service, Configuration Item.

    To learn more, see Nano Matching Panel
    Portal Settings

    Enable configuration options to streamline and simplify the Self Service Portal:

    • Hide the Shopping Cart to deliver a more streamlined experience.
    • Show a consolidated view of calls and requests as 'My Items', simplifying the way Users find and view their calls and requests.
    • Provide quick access to Promoted Catalog Items by displaying them on the home page

    Further configuration is required, see Portal Settings
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.
