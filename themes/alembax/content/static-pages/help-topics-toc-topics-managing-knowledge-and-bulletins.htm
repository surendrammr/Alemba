Managing Knowledge and Bulletins

This area of the documentation will deal with managing knowledge entries and bulletins to appear on the bulletin board.

Within the Bulletins section, you can find out about:

  • Creating bulletins
  • Updating the details
  • Attaching files
  • Selecting which groups can see the bulletin
  • Selecting which partitions can view the bulletin
  • Forwarding the bulletins
  • Making the bulletin available for viewing on the Self Service Portal
  • Deleting bulletins

You can configure the bulletin board using options within System Administration.

Within the Knowledge section, you can find out about:

  • Creating a knowledge entry
  • Moderating the entry to ensure it is accurate before it is published
  • Updating the details
  • Reviewing the knowledge entry
  • Deleting the knowledge entry
  • Searching for knowledge entries
  • Viewing the knowledge entry
  • Giving feedback on knowledge entries

You can configure the knowledge bank using options within System Administration.
