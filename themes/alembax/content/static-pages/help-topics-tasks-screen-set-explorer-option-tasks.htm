Screen Set Explorer Option

Where a number of different Task Details or Approval Details windows have been created through ASM Designer, the Screen Set explorer option enables you to change the task or approval screen, by selecting another screen set which contains your custom task/approval details screen.

Before you start

The Screen Set Explorer option is only displayed if Multiple Task Screen Sets is selected on the Workflow Management Settings window.

You must have access to more than one task screen set on the Task Screen Sets tab of your Workflow Management security role.

  1. In the Task Details window, select the Screen Set explorer option.
  2. In the Task/Approval Screen Set window, select another screen set. All currently defined task and approval screen sets appear.
  3. The Task Details window will refresh to use the new details window in the selected screen set.
  4. Any details that have already been added to the task or approval will be copied into the equivalent fields, where they are displayed on the new window. Even if these fields are not displayed on the selected screen, the information is stored in the database.
