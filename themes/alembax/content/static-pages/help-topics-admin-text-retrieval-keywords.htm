Keywords

You can maintain a list of keywords which can be attached to Knowledge Bank articles. Searches for articles can be based on these keywords. They are not case sensitive.

Adding a Keyword

Before you start

You must have Text Retrieval Setup selected in your own General Access security role before you can add a keyword.

  1. Select ≡ > Admin > System Administration.The System Administration window appears.

  2. In the Explorer pane, under Text Retrieval, select Keywords.

    A browse table of existing keywords appears.

  3. Select . A blank row is inserted in the browse table.
  1. Click to position your cursor within the new row and type the name of the new keyword you want to add to the list. Although the words are displayed in uppercase, you can enter them in lowercase.
  1. Repeat for each word you want to add to the list.
  2. Select to save the changes. Provide the Change Reasons if prompted to do so. The list is redisplayed, with the new word(s) in uppercase, filed in alphabetical order.

Renaming a Keyword

Before you start

You must have Text Retrieval Setup selected in your own General Access security role before you can rename a keyword.

  1. Select ≡ > Admin > System Administration.The System Administration window appears.
  2. In the Explorer pane, under Text Retrieval, select Keywords.
  3. In the browse table, select the word you want to edit.
  4. Overtype with the name you want to assign for the selected word.
  5. Repeat steps for each word you want to rename.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so. The list is redisplayed, with the word(s) in uppercase, filed in alphabetical order.

Deleting a Keyword

Before you start

You must have Text Retrieval Setup selected in your own General Access security role before you can delete a keyword.

You cannot reverse this procedure. If you deleted a word by mistake, you have to re-create the entry by adding the word to the list.

  1. Select ≡ > Admin > System Administration.The System Administration window appears.
  2. In the Explorer pane, under Text Retrieval, select Keywords.
  3. In the browse table, select the word you want to delete.
  1. Select .  The word disappears from the list, but the changes are not permanent until you save the changes.
  2. Repeat for each word you want to delete.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so. The list is redisplayed and the changes are permanent.
