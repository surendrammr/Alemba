Adding a Note in Self Service Portal

You can add notes to incidents, requests and approvals that are open. They are a useful function to add more information.

  1. Display the Incident, Request or Approval.
  2. Select the button at the top of the page.
  3. The Add a Note to your <Entity> page appears. The entity number is shown. Key the details into the Description field and then select to send an email or pager to the analyst to whom the incident/request/approval is assigned.
  4. Click OK. An acknowledgment appears.
