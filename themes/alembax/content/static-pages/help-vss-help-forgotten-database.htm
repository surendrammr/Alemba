Forgotten Database

The Database field in the Self Service Portal Login page is automatically updated by ASM Core. The database is specified within Self Service Portal URL.

http://win2k380.model.infra.com.au/infraEnterprise/infraEnterprise.aspx?LITE&BTN_FORGOTPASS=TRUE&ID=9227177878888888988888889812&DATABASE=ABC123

Where the name of the database is ABC123

If the database field is blank, contact your System Administrator to confirm the name of the database and enter this into Self Service Portal URL.
