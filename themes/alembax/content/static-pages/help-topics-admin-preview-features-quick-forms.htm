Quick Forms

This is a Preview Feature. Preview Features provide early access to new features which are still in development, and should not be enabled in Production environments.

Quick Forms are linked to Types Tiers and display a list of pre-defined questions in the Call Details screen when the linked Type is selected. Completing a question in the Quick Form advances the progress bar at the top of the Call Details window. Use Quick Forms to improve time to call resolution, ensure important information is captured, and support new staff while they get up to speed.

Enabling Quick Forms

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

    In the Explorer pane, expand Preview Features.

  2. Select Settings to display Preview Features Settings window.
  3. Select the check box for Quick Forms.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Configuring Quick Forms

Creating a Quick Form

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

    In the Explorer pane, expand Preview Features.

  2. Select Quick Forms to display the window.
  3. All existing Quick Forms are displayed. There are no forms on first time configuring the system.
  4. To create a new quick form, select beside Form Lists.
  5. A new row is added. Type a name for the quick form.
  6. To add questions to the quick form, select the form on the left, and then select beside List Items.
  7. A new row is added. Type the question in the row. This question will appear in the call details.
  8. Select the Is Require check box if the question's answer needs to be verified by the analyst.
  9. To change the question order, select a question and then press to move the question up or down
  10. Select to save the changes. Provide the Change Reasons if prompted to do so.

Linking a Quick Form to Type Tiers

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

    In the Explorer pane, expand Preview Features.

  2. Select Link Types to Quick Forms to view the window.
  3. Existing links between Quick Forms and Type Tiers are displayed. There are no links on first time configuration.
  4. Select a Type Tier from the list on the left, and then select a Quick Form from the list on the right.
  5. Press Link Type to Form to link the Type and Quick Form. The linked items appear in the Linked Types table.
  6. To remove a link, select the linked items in the Linked Types table and then press Remove Link.
  7. Select to save the changes. Provide the Change Reasons if prompted to do so.

Using Quick Forms in ASM Core

When Quick Forms are enabled and configured, the quick forms appear in the Call Details when a linked Type Tier is selected by the Analyst.

If there is more than one Quick Form linked to a Type Tier, the first form is displayed by default. Analysts can use the Quick Form field to select a different quick forms linked to that Type Tier.

Selecting a different quick form clears all answers from the previous form.

To complete a Quick Form the analyst types an answer for each question. Answers that require verification by the analyst have a check box beside them, which must be selected to complete the form.

A progress bar at the top of call details page shows the percentage of questions completed.

Answers are saved to the call when it is Saved, Deferred, Forwarded, or Closed.
