Viewing the Forward Schedule of Change in Self Service Portal

The Forward Schedule of Change, or Change Schedule, is a view that contains details of all the changes approved for implementation and their proposed implementation dates.

You can view the Forward Schedule of Change in Self Service Portal for requests that have been raised and are scheduled for implementation within a specified month.

When you navigate to this page, it appears with requests and change windows being implemented in the selected duration. The selected period has a colored background to indicate change windows. The dates with the underline represent days on which changes may occur. The dates with the dark, bold typeface represents the day on which the change implementation starts.

You must have Review Own Requests, Review Org Requests and/or Review All Requests selected in the Options tab of your Self Service Portal role, depending on your viewing requirements.

  1. On the My Options menu, select Requests > View Forward Schedule of Change .
  2. The Forward Schedule of Change window appears. The top half of the window displays a calendar with three months, with the current month in the center. You can move forward and back using the arrow buttons. You can also view changes scheduled for a specific day by selecting that day. The Legend beneath the calendar indicates the meaning behind the color and/or format of the date numbers
  3. You can filter the schedule details using the Filters fields.

    4. Type Use the drop-down list to identify the change type.
    Priority Identify the priority of the change using the drop-down list.
    Service Identify the service the change relates to. You can use the button to do a search if you wish.
    Config Item Identify the config item the change relates to. You can use the button to do a search if you wish.
