Controlling log in

You can list all the Analysts currently logged into ASM Core, as well as the license usage information for your system. You can also prevent people from logging in, if, say, you want to perform a backup. Likewise, you can log Analysts out, email Analysts, and access the details of an Analyst.

Preventing people from logging in

Use the Stop Logins option to prevent analysts logging into ASM Core, and Users logging into Self Service Portal. Analysts and Users already logged in will not be logged out.

Before you start

You can only enable and disable this option using the default Admin login.

  1. Select ≡ > Admin > System Administration. The System Administration window is displayed. In the Explorer pane, expand Security.

  2. Select Login Control.

    The Login Control window appears.

  3. Select the Stop Logins checkbox.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Logging out analysts

You can log Analysts out of the system if necessary. Person records that are identified as both Analysts and Users in their person details will be considered Analysts for this function. If a person record is identified as a User only, they will not appear in the list.

Before you start

You must have Security Setup selected in the Admin tab of your General Access security role to access the Security options within the System window.

  1. Select ≡ > Admin > System Administration. The System Administration window is displayed. In the Explorer pane, expand Security.
  2. Select the Login Control option from the Explorer pane. The Login Control window appears.
  3. To log one Analyst out of the system, highlight the name and select in the toolbar.
  4. To log out every Analyst and User who is currently logged in, select .

The Logout All Analysts option will log both Analysts and Users out of ASM Core and Self Service Portal, respectively.

Emailing analysts

Before you start

You must have Security Setup selected in the Admin tab of your General Access security role to access the Security options within the System window.

  1. Select ≡ > Admin > System Administration. The System Administration window is displayed. In the Explorer pane, expand Security.
  2. Select the Login Control option from the Explorer pane. The Login Control window appears.
  3. To email one or more analysts, select a required analyst, then select from the toolbar.
  4. To email all of the analysts listed in the browse table, select from the toolbar.
  5. In the Send Email window, complete the email details and select Send.

Viewing an Analyst's details

Before you start

You must have Security Setup selected in the Admin tab of your General Access security role to access the Security options within the System window.

  1. Select ≡ > Admin > System Administration. The System Administration window is displayed. In the Explorer pane, expand Security.
  2. Select the Login Control option from the Explorer pane. The Login Control window appears.
  3. Select the Analyst whose details you want to view and then select to display the Person Details for the Analyst.
  4. You will need to scroll down to see all the details. There is more information on creating and editing person details here.
  5. Select Ok or Cancel to leave the page.
