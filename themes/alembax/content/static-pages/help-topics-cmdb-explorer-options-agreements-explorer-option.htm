Agreements Explorer Option

The Agreements explorer option appears on the following panes:

  • CMDB Items:
    • Config Item
    • Service
    • Service Action
    • Service Bundle
    • Software Product
  • Person
  • Contract

This enables you to view all the agreements where the entity, person or contract is specified in the agreement selection criteria.

Before you start

This option is only enabled if the entity displayed on screen is used as a selection criterion in a service level agreement, in other words, there is at least one agreement with this entity specified in the IPK or Workflow agreement selection criteria.

  1. From the Details window, select the Agreements explorer option. This normally appears within the CMDBÂ Item group. Scroll up to locate the group, and expand it if necessary.
  2. Agreements that have this entity specified in the agreement selection criteria are listed in the browse table in the Linked Agreements window.
  3. To view the details of a linked agreement, select the entry in the browse table.
