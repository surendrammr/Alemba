Defining public holidays

You can create multiple public holiday calendars for your organization, and specify different public holidays for each calendar.

For example, you may want to separately define the public holidays for your US and UK offices.

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

Creating a public holiday calendar

Select â‰¡ > Admin > System Administration.

The System Administration window appears.

  1. In the Explorer pane, expand System, then select Public Holidays Definition to view the Public Holidays Definition window.

    The top panel contains a list of existing public holiday calendars.

  2. Select to create a new calendar row.

  3. Select , then type in a name for the new calendar.

    For example, Public Holidays US.

  4. Select alongside the row to save the name.

  5. Select in the toolbar to save the calendar.

Select alongside a calendar name to edit the name. Select alongside the row to save the changes. Select in the toolbar to preserve your changes to the calendar.

Setting a date as a public holiday

  1. In the Public Holidays Definition window, select the public holiday calendar where you want to define public holidays.

  2. Using the monthly calendar in the bottom panel, navigate to the required date.

    You can move between months using the up and down arrows at the top right of the monthly calendar.

  3. Select a date to designate it as a public holiday.

    Holidays are marked with a blue highlight.

  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

  5. Repeat for each date that you want to set as a public holiday.

To remove a public holiday, select the marked date. The blue highlighting is removed.

Copying a set of public holidays

You may find that you want to use an existing set of public holidays as a basis for another calendar, rather than creating the calendar from scratch.

  1. In the Public Holidays Definition window, select the calendar that contains the public holidays you want to copy.

    For example, select Public Holidays US.

  2. Select to copy the holidays within the calendar.

  3. Select the calendar where you want to paste the copied holidays.

    If the calendar does not yet exist, you need to create it.

  4. Select to paste the public holidays.

    You can now modify the holidays.

  5. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a public holiday calendar

You can delete all public holiday calendars, except the system-defined Default.

  1. In the Public Holidays Definition window, select the calendar you want to delete.

  2. Select to delete the calendar from the row.

    You must save to confirm the deletion.

  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
