Publishing a dashboard

You need to publish an existing dashboard in the Dashboard Server so that you can use it from within Alemba Service Managerâ„¢.

To publish a dashboard:

  1. Using the web designer, ensure your dashboard is open.

  2. On the title bar, select Publish.

    The Publish As Dashboard window appears.

  3. Select a Category from the drop-down list.
  4. Type a Name for the dashboard that will identify the dashboard in the Dashboards List.
