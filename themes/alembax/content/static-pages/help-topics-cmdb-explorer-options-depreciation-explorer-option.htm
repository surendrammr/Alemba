Depreciation Explorer Option - CMDB Item

Each CMDB Type can be linked to a depreciation type, method and rate (as percentage). The Depreciation explorer option enables you to view the depreciation information for the current CMDB Item.

Depreciation configurations are all based on CMDB Type, not on the individual item

This information is for reference purposes only. No actual depreciation calculations or write downs are performed against the CMDB item.

Viewing depreciation data for a CMDB item

Before you start

The depreciation configurations must be defined in the Depreciation by CMDBÂ Type window.

  1. From the CMDB Item Details Explorer window, expand the Data tab.
  2. Select Depreciation from the menu options. The Depreciation window appears.
