CTI Caller IDs Explorer Option

The CTI Caller IDs explorer option enables you to view or edit the telephone numbers stored for a person.

Adding a Telephone Number

Before you start

CTI must be selected in the System Settings to enable this option.

  1. From the Person Details window, in the Explorer pane, expand Linked Resources, then select CTI Caller IDs.

    The Caller Numbers window appears.

  2. You must be in edit mode to make any changes to the person details, so select from the toolbar if necessary to enable the other toolbar buttons.
  3. Select .
  4. In the highlighted row, type the User telephone number you want to add to the Telephone list.
  5. Repeat for each telephone number you want to add.
  6. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.

Modifying a Telephone Number

Before you start

CTI must be selected in the System Settings to enable this option.

  1. From the Person Details window, in the Explorer pane, expand Linked Resources, then select CTI Caller IDs.

    The Caller Numbers window appears.

  2. Select the telephone number entry you want to modify from the list by clicking on it. The entry is highlighted.
  3. Type the new telephone number details.
  4. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.

Deleting a Telephone Number

Before you start

CTI must be selected in the System Settings to enable this option.

You cannot reverse this procedure. If you deleted a telephone number by mistake, you have to re-create the entry by adding a telephone number.

  1. From the Person Details window, in the Explorer pane, expand Linked Resources, then select CTI Caller IDs.

    The Caller Numbers window appears.

  2. You must be in edit mode to make any changes to the person details, so select from the toolbar if necessary to enable the other toolbar buttons.
  3. Select the telephone number entry you want to delete from the list by clicking on it. The entry is highlighted.
  4. Select .
  5. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.
