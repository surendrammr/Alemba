Request Physical Statuses

The physical status of a request appears on the Info Panel of the Request Details window and on the Requests Outstanding window.

You can rename the statuses of the request, but you cannot add or delete request physical statuses.

Renaming a Request Physical Status

Before you start

You must have Workflow Management Set Up enabled within your General Access Security Role to rename a request physical status.

If you want to create or modify titles for describing the completion status of requests, use the Request Completion Statuses window.

  1. Select â‰¡ > Admin > System Administration. The System Administration window appears. In the Explorer pane, expand Workflow Management.
  2. Select the Request Physical Statuses option. The Request Physical Statuses window appears, displaying a browse table with the system defined Title for the status, as well as the Admin assigned Name .
  3. Key the name you want to assign into the Name field.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.
