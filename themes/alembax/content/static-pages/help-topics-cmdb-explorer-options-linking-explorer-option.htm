Linking Explorer Option

The Linking explorer option enables you to display a graphical view of the other CMDB items which are linked to the CMDB item. This is also referred to as the Linking Diagram.

Before you start

This option is only available if there are links between the current CMDBÂ item and another one.

  1. From the <Entity> Details window, select Linking from the explorer pane. This normally appears in the Linked Resources group.
  2. The CMDB Item Linking Diagram appears, showing the relationships between the current item and other items. Double click on any of the other items in the diagram to view their details.
