Request Search Explorer Option - CMDB Item

You can view open requests for CMDB items.

  1. From the <Entity> Details window, select Request Search from the explorer pane. This normally appears in the Linked Resources group.
  2. The Request Search window appears, with a browse table containing a list of all active requests associated with the current item.
  3. To view a request in more detail, select it and select .
