The New Option

The New option on the menu allows you to create different entities, such as CMDB Items, People, or Subscriber Groups.

  1. Select ≡ > New, then select from the following options:

    2. The menu options for parts of the application to which you do not have access are grayed out.

    Request Schedule Opens the Schedule Request from Workflow Template window, which enables you to schedule requests to be created at a future date.
    Knowledge Displays the New Knowledge Entry dialog box, which is the first step in creating a knowledge entry.
    Message Displays the Send Email window that you can use to send an internal and/or email message to selected analysts or groups.
    CMDB Item… Displays the New CMDB Item window that enables you to create a new configuration item, service, service action, or service bundle.
    Configuration Item Displays the Config Item Details window that enables you to create new configuration items.
    Inventory Displays the Inventory Item Details window that enables you to create new inventory items.
    Service Displays the Service Details window that enables you to create new services.
    Service Action Displays the Service Action Details window that enables you to create new service actions.
    Service Bundle Displays the Service Bundle Details window that enables you to create new service bundles.
    Software Product Displays the Software Product Details window that enables you to create new software licenses.
    Subscriber Group Displays the Subscriber Group window where you to create a subscription group and link it to Service Actions.
    Person Displays the Person Details window where you create a new User and link them to an organization or location.
    Contract Displays the Contract Details window where you can define a new contract and link it to an external supplier.
    Cost Center Displays the Cost Center window where you can define a new cost center and link it to people.
    Location Displays the Location Details window that you can use to define a new location.
    Organization Displays the Organization Details window where you can create a new organization and link it as a parent or child of other organizations you have defined on your system.
    Jurisdiction Displays the Jurisdiction Details window where you can define a new jurisdiction and link it to analysts
    Agreement Displays a sub-menu with the options SLA, OLA, and UC. You can use any one of these sub-menu options to open the Agreement Details window and create the selected type of agreement.
