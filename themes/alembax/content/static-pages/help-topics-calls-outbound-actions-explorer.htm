Outbound Actions Explorer Option

f the current call initiated an Outbound Action, you can view the outbound action messages exchanged between ASM and the external system relating to the Outbound Action.

Outbound Actions can trigger when a call is forwarded externally to a supplier and/or contract matching the details defined in an Outbound Action.

  1. Display the Call Details window, if it is not already on screen.
  2. In the Explorer pane, expand Data, then select Outbound Actions.
  3. The Outbound Events window appears. Select an option from the action navigator (to the left of the window)
    • Root node
    • Transaction History
    • Transactions
  4. View the action information in the browse table.
