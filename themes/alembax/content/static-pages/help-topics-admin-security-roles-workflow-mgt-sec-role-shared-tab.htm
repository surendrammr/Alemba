Workflow Management Security Role: Shared Tab

You can control access to security functions, change windows, and objects within Workflow Management using the Workflow Management security role.

Before you start

You must have Security Roles setup selected in the Admin tab of your own General Access security role before you can assign or remove permissions for any security roles.

  1. Display the Workflow Management Security Role details window, if it is not already on screen.

    Select â‰¡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Security Roles, then select Workflow Management.

    The existing Workflow Management security roles are displayed.

    The pre-defined role All typically has all role options selected. However, some of the options are not selected by default and you should explore the permissions set before assigning the role.

    If you want to create a role, select and assign the role a name in the Role Name field. To view or update a role, select the role in the browse table and select .

  2. Select the Shared tab .
  3. Choose the settings you wish to assign to the security role:
Workflow Analyst Analysts with this role can access to workflow management in ASM Core.

Make sure you have this option selected when setting up a Workflow Management role. Deselecting it will prevent the Analyst from participating in workflow management.

The other options on this window are only enabled if this option is selected.

Available Recipient Analysts with this role can have tasks and approvals forwarded to them.

They will appear on the list of available Analysts in the Forward Task Internally window.

If you deselect the option, the Analyst will not receive email messages for forwarded tasks or internal messages through the Bulletin Board
Budget Analysts with this role can fill in budget details on a request.
View Change Calendar Analysts with this role can search for and view change calendars

Change Calendar settings also apply to the Change Window explorer option in Requests.
New Change Calendar Analysts with this role can create new change calendars and add change windows linked to change calendars.

The previous option must be selected to enable this one. Change Calendar settings also apply to the Change Window explorer option in Requests.
Update Change Calendar Analysts with this role can edit change calendars. This option also enables the Edit button on the Change Calendar Details window.

The View Change Calendar option must be selected to enable this one. Change Calendar settings also apply to the Change Window explorer option in Requests.
Delete Change Calendar Analysts with this role can delete change calendars and change windows

The previous option must be selected to enable this one. Change Calendar settings also apply to the Change Window explorer option in Requests.
Cancel Change Calendar Lock Analysts with this role can override the lock that an Analyst has on a change calendar
Change Calendar Object View Analysts with this role can view objects attached to a change calendar

The View Change Calendar option must be selected to enable this one. Change Calendar settings also apply to the Change Window explorer option in Requests.
Change Calendar Object Update Analysts with this role can edit objects attached to a change calendar

The previous option must be selected to enable this one. Change Calendar settings also apply to the Change Window explorer option in Requests.
Change Calendar Object Cancel Analysts with this role can cancel check-out on objects attached to a change calendar. They can also cancel another Analyst's check out and check out the object themselves.

Change Calendar Object Version Control must be selected in Workflow Management Settings to enable this option. Change Calendar settings also apply to the Change Window explorer option in Requests.
Change Calendar Object Delete

Analysts with this role can delete objects attached to a change calendar.

Change Calendar settings also apply to the Change Window explorer option in Requests.
View Resource Manager

Analysts with this role can view the Resource Manager window, and manipulate task allocations within it.
Allocate Requests to Change Calendar Analysts with this role can set change calendars for CMDB items linked to a request
View Manage CMDB Transactions Analysts with this role can select and view Manage CMDB Task Transactions from a request and task
Edit Manage CMDB Transactions Analysts with this role can select and update Manage CMDB Task Transactions from a request and task
  1. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so. Alternatively, select another tab, if appropriate.
