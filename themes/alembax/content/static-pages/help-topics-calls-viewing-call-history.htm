Viewing Call History

You can view a summary of the current call history by selecting the History tab in the Call Details window. To view detailed call history information, select the History explorer option, or the button on the toolbar in the Call Details window.

The Call <number> History window appears with the call history for the current call.

The call history shows the actions carried out at previous stages. These include both the notes entered in the Actions and Solution field by the analyst, and automatic entries made by the system (such as, the times the call was actioned). The call history also displays information about Workflow rules that were run and those that were overridden by the analyst, details of any agreement escalation or breach on the call, calls or requests linked to the current call, and so on. It is not possible to change call history data once it has been recorded.

Two types of date are displayed on some history entries:

  • The Physical Date on the left-hand side which is indicated by a label with the same name. This is the database server date/time stamp at the time of the action on the call.
  • The user-defined date appears on the right-hand side. This is the database server date/time or the time set by the analyst taking action on the call.

Call action times are displayed as the local server time. This is regardless of the time zone in which the analyst is located.

If you have uploaded any objects, that action is also recorded in the history and a hyperlink to the object uploaded appears. Clicking this link opens the object.

Any system-generated history entries for a call will use an ‘Unspecified’ security profile.

By default, the history displays all actions on a call in ascending chronological order. However, the call description is always displayed first, regardless of the order selected.

The following buttons are available from this window:

To hide the explorer pane
To return to the main Details window
or To sort the order in which the events in the call history are displayed
or To expand or contract the details for all of the events in the call history. You can expand the details for different events using the button beside the event
To search for a text string within the call history
