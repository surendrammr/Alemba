Rebranding the Dashboard Server

You can customize the Dashboard Server to suit your company branding.

To customize the Dashboard Server:

  1. Using Dashboard Server, in the left-hand menu bar, select to open Settings.

  2. On the Site tab, configure the following general settings:

    Setting Description
    Organization Name

    Dashboard Server name that appears in the browser title bar and at the top of the default email template.

    Company Name Dashboard Server
    Site URL If required, you can change the Dashboard Server. Consult Support before you change this setting.
    Welcome note

    Text up to 70 characters long.

    Welcome to Company Name Dashboard Server

  3. Configure the following Images:

    Setting Description
    Log in screen We recommend this image is 240 x 120 pixels.
    Header Logo We recommend this image is 40 x 40 pixels.
    Favicon The favourite icon that appears alongside the browser title. We recommend this image is 16 x 16 pixels.
  4. Configure the Display settings as required.
  5. Disable Show powered by Syncfusion by moving the toggle to the left.
  6. Disable Show copyright information by moving the toggle to the left.
  7. Leave Show server version enabled to assist with support.
  8. Select Save to save your changes and update the settings.
  9. On the Dashboards tab, enable Mark dashboards and widgets as public by sliding the toggle to the right.
  10. Select Save to save your changes and update the settings.

  11. In the left-hand menu bar, select to open the Homepage.

  12. In the top-right corner select , then select Create > Category.

    The Add Category page appears.

  13. In Category Name, type Public Dashboards.
  14. Select Add to create the new category.
