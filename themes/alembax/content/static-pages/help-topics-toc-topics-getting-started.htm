Getting Started

With ASM Core, you have a powerful tool with which to run your service desk, from logging calls right through to workflow management, AI Ops and reporting. You can restrict access to functionality and data depending on the needs of the analysts and users, as well as keep everyone up to date with bulletins and knowledge articles.

To get started using ASM Core, explore some of the topics within this section. You can find out more about the basics of using ASM Core, including:

  • Using the online help
  • Logging in, changing your password and choosing your partition
  • Standard features that you will find throughout the system
  • Personalizing your system

You may also want to explore some of the other topics within this section to find out more about the Standard Features in vFire, and how to use them.
