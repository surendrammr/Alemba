About Call Activities

Call Activities enable you to create activity items that you can attach to a call, much like a checklist. Analysts can use these checklists as a step-by-step guide to ensure efficient call closure. Analysts can close the activity items as they complete them, which enables administrators to track call progress.

Administrators can reuse lists across multiple calls. The activity items within the lists are not linked between calls – if you close an item from within a call, it is closed for that specific call only.

You need the following permissions to configure and use call activities:

  • IPK Setup – for administration, which includes creating Call Activity Lists. For more information, see General Access Security Roles – Admin Tab.
  • IPK analyst – enables analysts to add activity call items to a call, and view and close call activity items within a call. For more information, see IPK Management Role: Options Tab.

Configuring Call Activities

You first need to create a Call Activity List before you can attach the activity items within it to a call.

To create a Call Activity List:

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

  2. In the Explorer pane, expand IPK Management, then select Call Activity Lists.

    The Call Activity Lists panel appears.

  3. Select to create a new Call Activity List.

  4. Type a name for the new list, then select to save the list.

To delete a Call Activity List:

  1. In the Call Activity List table, select the row you want to delete, then select .
  2. Confirm the deletion to remove the list from the table.

To add items to a Call Activity List:

  1. In the Call Activity Lists panel, select the list where you want to add call activity items, then select to view the Call Activities (Jen's note: I prefer Call Activity Items) table.

  2. Select to create a new item.

  3. Type a Description(Jen: I prefer Summary, see note) and Details for the item, then select to save the item.

To change the Call Activity item list order:

  1. View the Call Activities table where you want to change the item order.
  2. Select an item in the table, then select or to change its position.

To delete Call Activity items from a list:

  1. In the Call Activities table, select the row you want to delete, then select .
  2. Confirm the deletion to remove the item from the table.

Using Call Activities in a Call

From within a call, you can:

  • Add call activity items to the call using the Explorer pane. You can create new activity items for the call, or you can add all the items from a specific list.
  • View the activity item details.
  • Close an activity item when you have completed the activity.
  • View which activity items are open or closed for that call.

To create a new activity item for a call:

  1. Display the relevant Call Details window.

  2. In the Explorer pane, expand Data, then select Call Activities.

    The Call Activities pane appears.

  3. Select to create a new activity item.

    The Activity Details window appears. (Jen's note: should this be Activity Item Details window?)

  4. Type a Description(Jen: I prefer Summary, see previous note) and Details for the item, then select to save the item.
  5. If you want to immediately close the item, select a Closed Date and Closed Time, otherwise leave these fields blank.

  6. Select New to create a new activity item.

    The item appears in the Call Activities table.

To add all the items in a call activity list to a call:

  1. Display the relevant Call Details window.

  2. In the Explorer pane, expand Data, then select Call Activities.

    The Call Activities pane appears.

  3. Use the drop-down list to select the call activity list you want to add to the call, then select Add Activities from List.

    All items on the selected call activity list are added to the call and appear listed in the table.

To view an activity item details:

  • On the Call Activities pane, double-click a listed item to open it.

    The Activity Details window appears.

To close an activity item:

  1. On the Call Activities pane, double-click a listed item to open it.

  2. Select a Closed Date from the drop-down list, then select the time.

    Jen's note: Surely you click OK to confirm that you want to close this?

    The activity item closes. You cannot re-open it.

To filter open or closed activity items on the Call Activities pane:

  • On the Call Activities pane, select an option to filter the listed activity items according to status:

    Open – displays all open activity items

    Closed – displays all closed activity items

    All – displays all items, regardless of status

To view and filter current Call Activity items:

  1. Select ≡ > View > Call Activities.

    The Current Call Activities screen appears. (Jen: see note)

    The Call Activity Items screen appears, listing all items in the system.

  2. Select an option to filter the Call Activity items according to status:

    Open – displays all open activity items

    Closed – displays all closed activity items

    All – displays all items, regardless of status

  3. Double-click an item to view the Activity Details window.
