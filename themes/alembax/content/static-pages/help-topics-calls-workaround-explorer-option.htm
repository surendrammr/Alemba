Workaround Explorer Option

The Workaround explorer option enables you to record the details of any workaround found for the current call. Recording a workaround for the current call automatically links it to the call.

When a workaround is linked to a call, the Workaround explorer option displays an orange square next to it.

  1. Display the Call Details window if it is not already on screen.
  2. In the explorer pane, expand Resolution, then select Workaround.
  3. The Call Workaround window appears.
  4. In the Description field, type a detailed description of the workaround you provided for the call logged.
  5. Click Copy Workaround to Children to copy this workaround to any linked child calls. This is useful if similar calls are logged for the same issue.
