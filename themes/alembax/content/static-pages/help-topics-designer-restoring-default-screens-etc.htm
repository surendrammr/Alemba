Restoring Default Screens, Forms or Message Templates

If you have made changes to a screen, form, or message template and wish to restore the system default version, you can do so. However, you will not be able to undo the changes made by the restoration.

Restoring to default will override any customizations made by Alemba Professional Services. Contact your Alemba account manager if you are uncertain about whether your actions will impact these customizations.

Before you start

You must have Designer selected in your General Access Security Role.

To maintain an audit trail of changes made to screens, forms and message templates, Designer must be selected in the Auditing window.

  1. Display ASM Designer window if it is not already visible on screen.

  2. Find the screen, if it is not already displayed.
    1. Select the Find a Screen tab.

      2. The left pane contains an alphabetical list of the main entity types in ASM Core (such as, Agreement, Bulletin, Call, etc) with which screens are associated.

      The main pane has a set of filters across the top of the window, beneath which there is a browse table containing a list of screens, templates and forms, and their details.

    2. Select the entity type from the left pane. The browse table displays the configurable screens, templates and forms associated with the selected entity, with the first one highlighted. The number of screens associated with the selected entity is displayed in the bottom right hand corner of the window .
    3. Filter and sort the screens if you wish: To filter the number of entries displayed on the table, use the fields at the top of the window:
      4. Screen Name Type a name and select Find.
      Screen Set A drop-down list of all screen sets. Each entity type has a standard screen set which has the same name as the entity. For example, the default screen set for calls is Call Screen Set.

      Certain entity types, such as Organization, Person, Agreement and Location, only have default screen sets. Others, such as Call, Request, Task, CMDB item, Contract and Knowledge can have custom screen sets and these will appear on the list.

      Screen Design A drop-down list of available designs

      To sort the information, click on the required column heading. The entries are displayed in alphabetical order for that column.

    4. Select the screen, template or form you wish to restore, and select Open Screen at the bottom of the window.
    5. The window changes and if you look in the top left corner, you will see that the View Screens tab is now selected.

    6. You can select the Find a Screen tab again if you wish to select a different screen.

  3. Select the View Screens tab.
  4. Select Restore Default.
  5. Read the warning message and select Yes to confirm the restoration.
  6. Complete the change reasons if prompted to do so.
