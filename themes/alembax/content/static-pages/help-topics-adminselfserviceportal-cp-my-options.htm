My Options

You can create multiple Self Service portal menu panes, configuring the order of the menu item links and section headings, renaming them and regrouping as needed, as well as creating new menu links and section headings.

The steps are as follows:

  1. Choose/create the My Option menu, as described below.
  2. Use Designer to modify font, color and background.
  3. Assign the new My Option to the Self Service portal system.

Creating a My Option

Before you start

You must have Self Service Portal Setup enabled within your General Access Security Role in order to configure any Self Service administration settings.

 

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Self Service Portal.

  2. Select Self Service Portal My Options to open the window.
  3. A browse table displays all of the existing My Options. Default and Classic are provided as standard.
  4. To create new My Option Menu select Add beneath the browse table to create a new row. Click inside the new row and then type the name of the My Options item you want to create.
  5. Below the browse table, the following fields are displayed:
Display Name Enter a title for the menu pane that will appear in the Self Service Portal
Menu Behavior

Choose the behavior of nested lists when their section titles are selected

  • Expand and contract inserts child menu items below their section title
  • Pop out opens a tiered list to the right of the section title
Options

The Available Options list shows all options that can be displayed in the pane, and Selected Options lists those items that currently appear.

The to the left of some entries in the Selected Options list indicates that the item is a section heading and contains options within it.
Selected Option Details

The Label and URL of the item selected in the Selected Options pane.

  • Change the Label if required.

  • Type the URL of the page to open when this option is selected in the Self Service Portal. Learn more about Configuring URLs for Custom Options below. If creating a section title, leave this blank.

  • The URL field is grayed out and non-editable for all menu options except the Custom Option.

Creating and Updating Options

You can add and delete options that are displayed in your My Options menu, as well as changing the order in which they are displayed, and organising them into sections.

  1. To add an option at the top level, select it in the Available Options table and then select Add.
  2. To create a section title, select Custom Option in the Available Options table and then select Add. In the Selected Option Details overwrite its Label value with the title you want for the new section. Leave the URL field blank.
  3. To add an option so that it appears within a section, select the section title in the Selected Options table, then select the option you want to add from Available Options, and select Add Child. The section title will now show beside it, if it was not already there.

  4. If Add Child is grayed out, you have selected an option in Selected Options that contains a URL, and therefore is not a section title. Options can only be added as child items to section titles.

    You can create several layers of submenu options, by creating sections as children, and then further sections as children of this submenu.

  5. To reorder the display of the options, use the Move Up and Move Down buttons.
  6. To remove an option from the menu, select Remove.
  7. To restore the default settings, select Restore.
  8. To create new menu options, select Custom Option from the left pane and add it to the right pane, then complete its details in the Selected Option Details settings. Learn more about Configuring URLs for Custom Options below.
  9. Select to save the changes. Provide the Change Reasons if prompted to do so.

If you have permission to access the Designer through your security role, you can change the look of the My Options menu pane by redesigning its features in the Skins tab of Designer.

Configuring URLs for Custom Options

URLs can be a link to a page within the Self Service Portal, such as to a specific knowledge article, or to a web page outside of it such as a third-party homepage.

To link to pages outside the Self Service Portal you must enter the full address beginning with www, http, or https.

To link to pages within the Self Service Portal, you do not need the full address in the URL, you only need to include the portion of the URL that follows ?LITE.

For example the full address for the Self Service Portal Knowledge Search page is http://<server>/<virtual_directory>/Core.aspx?Lite&Form=KnowledgeSearchCriteria so you would enter &Form=KnowledgeSearchCriteria into the URL field.

See URLs and Access Parameters to learn more.

Updating a My Option

Before you start

You must have Self Service Portal Setup enabled within your General Access Security Role in order to configure any Self Service administration settings.

 

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Self Service Portal.

  2. Select Self Service Portal My Options to open the window.
  3. A browse table displays all of the existing My Options. Default and Classic are provided as standard.
  4. Select the My Option menu you want to update.
  5. Make the necessary changes.

Deleting a My Option

Before you start

You must have Self Service Portal Setup enabled within your General Access Security Role in order to configure any Self Service Portal administration settings.

You cannot delete the system defined My Options titled Default and Classic.

Once deleted, My Options cannot be restored.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Self Service Portal.

  2. Select Self Service Portal My Options to display a browse table of existing My Options.

  3. Select the My Option you wish to delete and select Delete. If the button is grayed out you have selected a system defined option which cannot be deleted and must select a different option to delete.

    A warning dialog box appears for you to confirm your choice.

  4. Select Yes to continue or No to cancel.
  5. The deleted My Option disappears from view, but it is not yet deleted from the system.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so.
