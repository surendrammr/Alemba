Installing the ASM app

  1. Download the app from the appropriate app store.
  2. Install the app following the onscreen instructions.

Once the app has been installed onto your device, you can configure its settings.

Configuring the ASM app

You will need to configure your app before using it for the first time. Once you have the settings in place, you will not need to do it for further logins on the same device.

If the ASM app is removed from your device or if you want to use the app on a new device, you will need to complete this configuration process again.

  1. From the login screen, select the menu button. This will differ depending on the type of phone.

    2. iOS Tap on the button in the bottom left hand corner
    Windows Tap on the button in the bottom right hand corner
    Android Tap on the button in the top right corner
  2. Select Settings to display the settings screen. (When you first log in, this screen appears automatically.)
  3. Enter the Server Name and System Name , and select the HTTPÂ settings, all of which should be specified by your system administrator.
  4. Select Save and close the screen.
  5. You can now log in to the application. The settings that you have entered will be validated when you log in.
