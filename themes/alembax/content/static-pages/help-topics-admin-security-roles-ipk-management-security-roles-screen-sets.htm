IPK Management Security Role: Screen Sets Tab

You can control access to specific call screen sets for the selected IPK role. You can also define a default screen set for the security role, so that when the Analyst logs a call, the Call Details window is automatically displayed using the default screen set.

Before you start

You must have Security Roles setup selected in the Admin tab of your own General Access security role before you can assign or remove permissions for any security roles.

The Screen Sets tab on the IPK Management Security Role window only displays when Enable Call Screen Set on Explorer is selected and Enable IPK Statuses and Streams is not selected in the IPK Settings.

Screen sets must first be defined in the Call Screen Set option in the IPK Management group.

  1. Display the IPK Management Security Roles details window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Security Roles, then select IPK Management. All existing IPK Management security roles are displayed.

    The pre-defined role All typically has all role options selected. However, some of the options are not selected by default and you should explore the permissions set before assigning the role.

    If you want to create a role, select and assign the role a name in the Role Name field. To view or update a role, select the role in the browse table and select .

  2. Select the Screen Sets tab .
  3. To add screen sets and make them available to the security role, select the screen set in the Select Screen Set table and select Add to List. The screen set appears in the Selected Screen Set table.
  4. To set a screen set as the default, select it in the Selected Screen Set table and select Set as Default. This means that when an Analyst logs a call, this screen set appears. Otherwise, they have to select from the list of available screen sets before the Call Details window appears.

    5. Make sure that Default Screen Set is selected on the Options tab.

  5. To remove a screen set, select it in the Selected Screen Set table, then select Remove.
  6. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so. Alternatively, select another tab, if appropriate.
