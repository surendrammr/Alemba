Request Details Information Panel

The Request Information Panel appears on the right side of the Request Details window, and it displays a summary of the field values relating to the currently displayed request. You may need to expand it to see the details.

The information is grouped into:

  • Request Information
  • Request Responsibilities
  • Request Dates

The values displayed on the Information Panel are automatically updated by the system, and as such cannot be edited.
