Global Search

This is a Preview Feature. Preview Features provide early access to new features which are still in development, and should not be enabled in Production environments.

The global search bar in Self Service Portal enables Users to easily search across Knowledge, the Service Request Catalog, calls, and requests.

Enabling Global Search

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

    In the Explorer pane, expand Preview Features.

  2. Select Settings to display Preview Features Settings window
  3. Select the check box for Global Search.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Configuring Global Search

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

    In the Explorer pane, expand Preview Features.

  2. Select Global Search to display the window.
  3. Enter a value in Max Display Results field to change the number of results to display in the Self Service Portal Global search Results Page.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Using Global Search in the Self Service Portal

When Users log into the Self Service Portal they see a Global Search bar below the banner.

To perform a global search Users type text into the search bar and press Enter. The Search Results window is displayed.

Users can filter the results by selecting and de-selecting filter options at the top of the page.

Selecting the title of a returned item loads the relevant window for that item:

  • Selecting a Service Request Catalog item to opens the corresponding submission form
  • Selecting a Knowledge Article opens the knowledge details
  • Selecting an Incident opens the Incident Review page
  • Selecting a Request opens the Request Review page
