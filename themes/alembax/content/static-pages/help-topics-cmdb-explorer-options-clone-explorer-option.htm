Clone Explorer Option

This explorer option enables you to copy a CMDB item in order to create a new one.

Before you start

You must have CMDB Item Clone selected in the CMDB Items tab of your Configuration Management security role.

  1. In the Details window, in the Explorer pane, expand Data, then select Clone.
  2. The Clone CMDB Item dialog box appears. Complete the details.
  3. Select OK to save the changes.
