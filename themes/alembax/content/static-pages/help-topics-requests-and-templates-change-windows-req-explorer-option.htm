Change Windows Explorer Option for Requests

The Change Windows Explorer option enables you to view and/or add Change Windows for CMDB items linked to the current request.

Before you start

You must have the appropriate Change Calendar permissions selected in the Shared tab of your Workflow Management security role.

Â 

  1. Select the Take Action button on the Request Details window if necessary to allow you to work on the request.
  2. In the Explorer pane, expand Data, then select Change Windows to display the Change Windows window. This window displays all the Change Windows which are applied to CMDB items (such as services and configuration items) linked to the current request.
  3. To apply a Change Window to a CMDB item linked to the current request, select Add New.
  4. In the Change Window pop-up, complete the details.

    5. Linked Item

    In the drop-down list, select the CMDB item to which you want to apply a Change Window.

    This list only displays the CMDB items linked to the current request.
    Change Calendar

    In the drop-down list, select the name of the Change Calendar which contains the Change Window(s) you wish to apply to the linked CMDB item. The values in the Change Window list are automatically updated with the names of the Change Windows belonging to the selected Change Calendar.

    The list only displays the Change Calendars linked to the CMDB item selected in the Linked Item field.

    Change Window

    In the drop-down list, select the name of the Change Window you wish to apply to the linked CMDB item.

    The dates and times of the Start Date and End Date fields are automatically updated to display those specified for the selected Change Window. However, you can change these date values if necessary.
    Start Date and End Date

    Select different start and end date/times for the change window if necessary.

    You must select dates within the change window.
    Update Request Implementation Start Date Select if you want ASM Core to automatically change the start date of the request to that of the Change Window.
    Update Request Implementation End Date Select if you want ASM Core to automatically change the end date of the request to that of the Change Window.
  5. Select Save to save the changes.
  6. When a Change Window is applied to a request, a small orange square appears next to the Change Windows Explorer option on the Request Details window.
  7. To view the details of a Change Window, select it in the browse table and click Open.
  8. To delete a Change Window, select the corresponding row on the browse table and click Delete.
  9. To show or hide the columns in the browse table, click Filter Columns.
