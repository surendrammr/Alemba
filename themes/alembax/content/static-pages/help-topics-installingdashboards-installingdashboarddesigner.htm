Installing Alemba Dashboard Designer

Use the desktop Alemba Dashboard Designer to design multi-tabbed dashboards that display information from the ASM database.

We recommend that you use the Web Designer if you want to present dynamic dashboard data that is relevant to the logged in user.

Before you start

Before you install Alemba Dashboard Designer:

  • Ensure you have installed and configured the dashboard platform. For more information, see Installing the dashboard platform.
  • If you install Alemba Dashboard Designer on a local machine, ensure that you can connect to the Dashboard Server via the correct protocol. For more information, see Configuring the web protocol.

To install the Alemba Dashboard Designer:

  1. Extract the installation package to the Dashboard Application Server.

    We recommend you extract the installation package to a folder on your c:\ drive. For example: c:\Temp

  2. Using Windows Explorer, right-click AlembaDashboardDesignerInstall.bat, then select Run as administrator on the shortcut menu.

    The installer installs the Alemba Dashboard Designer, deploys PhantomJS and CefSharp, performs branding, and creates shortcuts.

  3. Launch the Alemba Dashboard Designer application from the desktop shortcut or Start Menu.
