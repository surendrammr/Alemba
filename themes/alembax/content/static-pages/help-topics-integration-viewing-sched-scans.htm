Viewing Scheduled Scans

You can display a log of the events for the scheduled scans that ran on a selected source to import external resources or users.

Before you start

The scheduling details for a source must have been configured.

You must have Integration Setup selected in the Admin tab of your General Access Security Role.

  1. Select â‰¡ > Admin > Integration.
  2. In the Explorer pane, under Integration, select Scheduling to view the scan log.

Any errors and warnings during scanning are logged in the Integration Platform Activity Log.

You can set up import rules during a scan when configuring the resource mapping details for a resource defined on the source.

You can set up integration scans to run on the source by defining the scheduling.
