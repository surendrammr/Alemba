Components Explorer Option

The Components explorer option enables you to display a graphical view of the components for a container CMDB item.

Components are not shown directly on the CMDB linking diagram. For more information on the components diagram which is very is similar to CMDB linking diagram, see the topic on the CMDB Linking Diagram.

Viewing the Components Diagram

Before you start

This explorer option is only enabled if the current CMDB item uses a Container-Components link and is the Container.

  1. From the Explorer pane in the Details window, select Components. This normally appears in the Linked Resources group. Expand the group if necessary.
  2. The CMDB Item Components window appears, giving you a graphical display of the components of the current item.

If there is an outage currently being simulated on the CMDB linking diagram, and an Analyst drills down into a component, the Components window appears with the outage simulation still running.
