Linking Approval Screens to Partitions

You can link approval screens that have been created using ASM Designer to partitions. This ensures that the approval screen is visible only to those analysts who are in that partition and have access to the approval screen through their Workflow Management Security Role.

  1. Select â‰¡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Workflow Management.

  2. Select the Approval Screens option. The Approval Screens window appears, displaying a browse table with two columns: the first column displays the name of all the approval screens created for your system, and the second one the names of all partitions created on your system.
  3. Locate the name of the approval screen you want to link.
  4. Click in the corresponding Partition column.
  5. From the list of partitions, select the partition you want to link to that approval screen.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so.
  7. If you want to change the linked partition, simply select a different one, and save your changes.
