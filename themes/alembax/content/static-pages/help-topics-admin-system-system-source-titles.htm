System Source Titles

Whenever a call, request or task is logged, ASM Core makes a record of how it came into the system, such as through the main application, the Self Service Portal, or an API transaction. This information is known as the System Source, and the name given to the source is the System Source Title. You cannot add or delete system source titles, but you can specify your own title for any system source.

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

  2. In the Explorer pane, expand System, then select System Source Titles. The System Source Titles window appears, listing all the sources through which a call, request or task can be logged.
  3. Select the title you want to rename, then select .
  4. Update the name.
  5. Select to save the changes. Provide the Change Reasons if prompted to do so. The list is redisplayed, showing the new source title in alphabetical order.
