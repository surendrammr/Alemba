Quick Notes

Quick notes are brief notes that Analysts can use when logging/actioning a call, request or task. Quick Notes added in the Quick Notes option are available for the partition you create them in.

Viewing Quick Notes

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

  2. In the Explorer pane, under System, select Quick Notes.

    The Quick Notes window appears, listing the quick notes that are currently available.

  3. Active is selected by default and displays all of the active quick notes. To display deleted ones, select the Deleted button.

Adding a Quick Note

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

You can add a quick note that can be used when logging/actioning a call, request or task.

  1. Select ≡ > Admin > System Administration. The System Administration window appears.

  2. In the Explorer pane, under System, select Quick Notes.

    The Quick Notes window appears.

  3. Select Add to add a new note.
  4. Complete the details as follows:

    5. Name Type the name of the note, up to 80 characters. This will appear in the drop down list of available Quick Notes from within the call, request or task window
    Call/Request/Task Select the entity for which the note is being created
  1. In the lower Quick Note text area, type the details to be included in the note.
  2. Select Visible in Self Service Portal to make the quick note available to Analysts and Users logging in via the Self Service Portal.
  3. Select a Security Profile to restrict the Quick Note to Analysts with a specific security profile. Leaving the default “Unspecified” will make it available to all Analysts.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Updating a Quick Note

You can change the details of a quick note to reflect changing needs of the organization/department.

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

  1. Display the Quick Notes details window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    Select Quick Notes from the Explorer pane. The Quick Notes window appears, listing the quick notes that are currently available.

  2. Select the quick note you want to update. The details are displayed in the Quick Note text area.
  3. Overwrite with the necessary changes.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.
