Connectors to ASM Core

There is a range of connectors that act as an interface between the ASM Core Integration Platform capabilities and any third-party vendor application, including the following:

Connector Name Third Party Application
Alemba® Email Event Connector Email applications
Alemba® External Process Connector An external process, such as PowerShell or other executable
Alemba® Stored Procedure Connector A stored procedure in the ASM database
Altiris Connector Altiris Network Discovery
Atlassian JIRA Connector Atlassian JIRA
Centennial Connector Centennial Discovery / FrontRange
Cohesion Connector CA Cohesion
CSV Connector A plain text Comma Separated Value (CSV) file
Database Resources Connector External databases
MS SQL Table Connectors SQL Server database tables
HP Systems Insight Manager Connector HP Insight
Ionix ADM Connector VMware vCenter Application Discovery Manager (ADM)
Ionix NCM Connector EMC Smarts Network Configuration Manager (NCM)
Ionix SAM Connector EMC Smarts Service Assurance Manager (SAM)
LANDesk Connector LANDESK
Novell eDirectory Connector Novell eDirectory
MS Active Directory Connector Microsoft Active Directory (AD)
MS Azure Connector Microsoft Azure Resource Manager
MS Azure DevOps Connector Microsoft Azure DevOps
MS SCOM Connector Microsoft System Center Operations Manager (SCOM)
MS SCCM/SMS Connector Microsoft System Center Configuration Manager (SCCM)
SolarWinds Connector SolarWinds systems (NPM, VM, SAM)
Sun Java System Connector Sun Java Directory Server / Oracle Directory Server
vCenter Orchestrator Connector VMware vCenter Orchestrator (VCO)
vCloud Director Connector VMware vCloud Director
ASM to ASM Connector ASM Core
VMware Configuration Manager Connector

VMware vCenter Configuration Manager (VCM)

Refer to the ASM Connector Matrix for more details of the versions that are currently supported.

You should familiarize yourself with the information in Installing Connectors before installing any connectors, and read the Integration topics for more information on how to configure them.
