Self Service Portal Display Categories

You can define Self Service portal display categories that appear in the list on the Service, Service Action, and Service Bundles Details windows in ASM Core, and when a user browses the Service Request Catalog in the Self Service Portal.

Creating a Portal Display Category

Before you start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings. Depending on if/how your system is partitioned, ensure that you are in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Self Service Portal Display Categories to open the window.
  1. Select Add New in the topmost tier.
  2. Type the name of the category in the Tier 1 field.
  3. To add lower tiers linked to this tier, first select the tier.
  4. Select Add New in the table for the next tier, and type the required value.
  5. You may wish to hide items in lower tiers from view in the Self Service Portal. You can do so by selecting the Hide Child checkbox next to the category. This will automatically select all of the items in the lower tier. However, if you only want to hide some of the items, you can deselect those that you want to make visible.

    6. You may want to use this feature if, say, all of the items in lower categories are automatically ordered as part of an item in a higher category, and you do not want them to be selectable on an individual basis.

  6. Select to save the changes. Provide the Change Reasons if prompted to do so.

Renaming a Portal Display Category

Before you start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings. Depending on if/how your system is partitioned, ensure that you are in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Self Service Portal Display Categories to open the window.
  1. Select the category entry you want to rename.

    2. If you want to rename an entry in a lower tier, ensure that you have selected the correct entry in the higher tiers to display the entry.

  2. Type the name you want to assign for the selected Portal Display Category.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a Portal Display Category

Before you start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings. Depending on if/how your system is partitioned, ensure that you are in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Self Service Portal Display Categories to open the window.
  1. Select the category you want to delete.
  2. Select Delete. You cannot delete the default entry. If you select it, the Delete button is grayed out.
  3. If you try to delete a category that has linked categories, the system will display a message that the selected category has linked tiers and the linked tiers will also be deleted . Select Yes to delete the selected category and its linked categories or No to close the message without deleting any values.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.
