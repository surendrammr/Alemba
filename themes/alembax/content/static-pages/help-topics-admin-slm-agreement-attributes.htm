Agreement Attributes

You can define categories of information related to an agreement such as User reference, organization name and so on. Analysts can then access and record information in these categories from the Agreement Details window. For example, if you create a ‘User Reference’ attribute, Analysts can add the reference number(s).

When you create an attribute, the related agreement table is updated with the Attribute field. When an Analyst adds values for an agreement attribute, the tables are updated with the attribute values.

Creating Agreement Attributes

Before you start

You must have SLM Setup enabled within your General Access Security Role in order to configure any CMDB administration settings.

The Agreement Attributes option will only display if the Attributes option is checked in the System Settings window.

  1. Select the Menu button , then Admin, and then select System Administration. The System Administration window is displayed, with a menu of options available . In the Explorer pane, select Service Level Management.

  2. Select Agreement Attributes to open the window.
  1. Select Add for the relevant tier. A new field is added to the list.
  2. Type the attribute name into the field.
  3. Define additional tiers by highlighting the first tier and then selecting Add in the lower tiers.
  4. Repeat these steps for each tier added.
  5. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting Agreement Attributes

Before you start

You must have SLM Setup enabled within your General Access Security Role in order to configure any CMDB administration settings.

The Agreement Attributes option will only display if the Attributes option is checked in the System Settings window.

If you delete an agreement attribute, any associated attribute in a lower tier will be deleted. If you delete an existing attribute, any details already defined for that category by the Analyst will be retained. However, Analysts cannot specify new values for this category.

  1. Select the Menu button , then Admin, and then select System Administration. The System Administration window is displayed, with a menu of options available . In the Explorer pane, select Service Level Management.

  2. Select Agreement Attributes to open the window.
  1. Highlight the Attribute that you wish to delete.
  2. Select Delete
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
