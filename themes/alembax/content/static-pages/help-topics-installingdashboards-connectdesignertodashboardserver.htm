Connecting Alemba Dashboard Designer to the Dashboard Server

To publish dashboards, widgets and data sources to the Dashboard Server, you must first connect Alemba Dashboard Designer to the Dashboard Server.

To connect Alemba Dashboard Designer to the Dashboard Server:

  1. Using Alemba Dashboard Designer, select User in the top right corner.
  2. Select New User Login from the drop down menu.

  3. In Server URL, type the Dashboard Server URL: http://ServerName/alembadashboardserver, where ServerName is the server name or IP address.

    For example: http://localhost/alembadashboardserver

  4. Type the Dashboard Server credentials.

    For example, AlembaAdmin.

  5. Select Login to connect to the Dashboard Server.

    The Dashboard Server user name appears in the top right corner.
