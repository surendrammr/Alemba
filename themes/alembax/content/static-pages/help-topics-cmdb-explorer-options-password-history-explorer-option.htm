Password History Explorer Option

The Password History explorer option enables you to view the history of changes to passwords for the person.

  1. From the Person Details window, in the Explorer pane, expand Data, then select Password History.
  2. The Person Password History window appears, showing the dates on which the person's password was changed, and whether they were changed by the person or the system administrator.

    3. You cannot display the password itself.

  3. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.
