Outbound Actions Explorer Option - Task

If an Outbound Action is triggered on a task, you can view the outbound action messages exchanged between ASM and the external system relating to the Outbound Action. Outbound Actions can trigger when an Outbound Action Task is activated in a workflow, or when an External Supplier task is forwarded to a supplier and/or contract associated with the Outbound Action.

  1. In the Task Details window, expand the Data tab on the explorer pane.
  2. Select Outbound Actions. The Outbound Events window appears.
  3. Select an option from the action navigator (to the left of the window):
  • Root node
  • Transaction History
  • Transactions
  • View the action information in the browse table.
