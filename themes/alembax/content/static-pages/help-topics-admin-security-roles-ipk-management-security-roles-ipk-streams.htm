IPK Management Security Role: IPK Streams Tab

Before you start

You must have Security Roles setup selected in the Admin tab of your own General Access security role before you can assign or remove permissions for any security roles.

This tab is only shown if you have Enable IPK Statuses and Streams selected and on the IPK Settings window. Otherwise you will see the Screen Sets tab. IPK streams must first be defined .

You can assign specific streams to security roles. Any Analyst who has been assigned the role will be able to log calls using the selected streams and will be able to log calls for those streams. Analysts will still be able to search for and action calls that have been logged with IPK Streams to which they do not have access through their role. When a default IPK Status and IPK Stream are set for a role, an Analyst with this role can simply click the Call button to launch the Call Details window linked to the default status and stream.

You must ensure that the role has access to streams and IPK Statuses that are linked. If not, Analysts with this role will not be able to log a call.

 

  1. Display the IPK Management Security Roles details window, if it is not already on screen.

     

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Security Roles, then select IPK Management. All existing IPK Management security roles are displayed.

    The pre-defined role All typically has all role options selected. However, some of the options are not selected by default and you should explore the permissions set before assigning the role.

    If you want to create a role, select and assign the role a name in the Role Name field. To view or update a role, select the role in the browse table and select .

  2. Select the IPK Streams tab .
  3. Select IPK Stream Security to enable the ability to set access to particular IPK streams. To allow access to all streams, deselect the option.
  4. To add an IPK stream, select it in the Select IPK Stream table and then select Add to List.
  5. To remove an IPK stream, select it in the Selected IPK Stream table, and select Remove.
  6. To set a default IPK stream, ensure that the default IPK Status is linked to IPK Streams. The Default IPK Stream/Status option in the Options tab must also be selected for this role.
  7. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so. Alternatively, select another tab, if appropriate.
