Forms Explorer Option for Requests

The Forms Explorer option enables you to attach a form to the current request. Forms are specially designed to capture general, non-reportable, non-searchable information about a request. ASM Core is provided with a default (blank) form for requests. You can use the ASM Designer to add fields to the default form or create new forms.

Before you start

The option is only enabled if Forms are enabled on the System Settings window.

You must have access to the form in the Forms tab of your Workflow Management Security Role.

  1. Select the Take Action button on the Request Details window if necessary to allow you to work on the request.
  2. In the Explorer pane, expand Data, then select Forms. The default form and any user-designed form are listed.
  3. Select the form you want and complete the details.
  4. Select to return to the Request Details window.
  5. The Form details are saved when you save the request details.
