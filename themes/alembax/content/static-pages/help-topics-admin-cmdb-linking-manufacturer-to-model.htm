Link Manufacturer to Model Type

You can link model types to specific manufacturers so that when a manufacturer is selected on the CMDB Item Details window, the list of available models is filtered according to manufacturer.

Linking a Model to a Manufacturer

Before You Start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings.

Manufacturer to Model must be selected in the CMDB Settings window to enable the Link Manufacturer to Model Type menu option.

Before you can link Manufacturers to Models, you need to have defined Model Types and Manufacturers.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Link Manufacturer to Model Type to open the window.
  1. From the Select Manufacturertable, select a manufacturer. The model types currently linked to that manufacturer are displayed in the Selected Linked Model Types table at the bottom of the window.
  2. From the Select Model Types table, select the model you want to link to the manufacturer.
  3. Select .
  4. Repeat for each model you want to link to a manufacturer.
  5. Select to save the changes. Provide the Change Reasons if prompted to do so.

Unlinking a Model from a Manufacturer

Before You Start

You must have CMDB Setup enabled within your General Access Security Role in order to configure any CMDB administration settings.

Manufacturer to Model must be selected in the CMDB Settings window to enable the Link Manufacturer to Model Type menu option.

Before you can link Manufacturers to Models, you need to have defined Model Types and Manufacturers.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select Link Manufacturer to Model Type to open the window.
  1. From the Select Manufacturer table, select a manufacturer. The model types currently linked to that manufacturer are displayed in the Selected Linked Model Types table at the bottom of the window.
  2. From the Selected Linked Model Types table, select the model you want to unlink from the manufacturer. The entry is highlighted.
  3. Select . The model is removed from the Selected Linked Model Types table.
  4. Repeat for each model you want to unlink to the selected manufacturer.
  5. Select to save the changes. Provide the Change Reasons if prompted to do so.
