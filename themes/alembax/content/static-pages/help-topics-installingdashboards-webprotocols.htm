Configuring the web protocol

  • Using the http drop-down, you can change the protocol of any URL on either the Dashboard Server or the User Management Server. You must first configure this in User Management Server.

    If you choose to use https, ensure you change each URL to use the same protocol in both the Dashboard Server and the User Management Server.

    Our examples use the http protocol.
