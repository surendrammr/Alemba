Self Registration

To access Self Service Portal, your details must be registered on ASM Core. A registered user has a Login ID and Password that can be verified by Self Service Portal.

To register your details, follow these steps:

  1. From Self Service Portal Login page, select the link to register in the top left corner.
  2. The Register page appears. Complete the details.

    3. The names of some of these fields may differ in your system. Mandatory fields are indicated by a red asterisk (*) against the field. You cannot proceed without typing information into the mandatory fields.

  3. Specify your personal details in the Title, First Name, Surname, Work Phone and Mobile fields.
  4. Type the email address with which you want to register for Self Service Portal in the Email Address field.
  5. From the Preferred Email Format list, select an option from Plain Text and HTML.

    6. When typing into some text fields in ASM Core, you can format your text as either Plain Text or HTML, depending on the default format chosen in system settings. If you are working in Plain Text, you cannot format your text in any special way. The text box appears without any formatting buttons or options. If you are working in HTML format, a formatting toolbar will appear above the description and message text area.

  6. HTML email messages support text formatting, numbering, bullets, alignment, image attachment, tables, HTML styles, stationery, signatures, and linking to Web pages. Because the most popular e-mail programs use HTML, it is the recommended format for Internet mail. Plain Text is used to specify the description when uploading objects, regardless of whether you are attaching objects to an HTML Knowledge entry or Bulletin Board item. When working with knowledge entries, plain text makes extensive use of knowledge article templates in which elements such as question, synopsis and answer are entered into fields. ASM Core then applies its own rules to render the article based on these fields.

  7. Type in your preferred Login ID in the Login ID field. This is the Login ID you will have to use to log on.
  8. Type in your preferred password in the Password field. Passwords are case-sensitive. You will need to enter this when you log in.
  9. Re-type your preferred password in the Confirm Password field. This has to be the same password as the one you typed in the password field.
  10. Click OK to register your details. A confirmation appears to acknowledge that your details have been successfully submitted.
  11. An email will be sent to you to confirm details of your registration, at which point you can log in to Self Service Portal.

    12. Depending on the configuration for Self Service Portal set by your System Administrator, your registration details may require to be reviewed by an administrator before a notification email is sent to you.

Troubleshooting

Email Address already in use, please enter a new address. You may already be an existing registered user. To retrieve your Login ID and Password, click the link in the top menu bar.
Login ID already in use, please enter a new Login ID.

You are an existing registered user or the Login ID you specified has already been registered by another user.

If you are an existing registered user and have forgotten your password, retrieve it using the link in the top menu bar.

If you are a new user, type in an alternative Login ID to proceed.

Password and Password Confirm do not match. Re-type your password in the Password and Password Confirm fields. Make sure they are identical, noting that passwords are case sensitive.
