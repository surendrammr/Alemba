Catalog Action Explorer Option

This explorer option is only available for service actions. It allows you to configure the display settings for the catalog portal.

  1. Display the CMDB Item Details window, if it is not already on screen.

    Search for the CMDB item. Then select the item you want to work with and select .

  2. Expand the CMDB Item group in the explorer pane if necessary, and select Catalog Action.
  3. Complete the details as required.

    4. Display as Config Item Action Select to display the service action as a config item action to display all items of that CMDB Item Type in the Catalog Portal. You will then need to specify the config item type in the list below.
    For Config Item Type Select the config item type from the multi tiered drop down list.
    Hide in Service Request Catalog Select this checkbox to hide the selected service actions from view in the Service Request Catalog.
  4. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.
