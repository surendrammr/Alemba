Searching for a User

You can search for and view user details in Self Service Portal.

The search enables you to search for all users who are linked to your organization and for users who are a part of other organizations which are linked to your organization (users within your organization tree).

If you are not linked to an organization, you will not be able to search for users.

  1. From the My Options menu, select User Search.
  2. The User Search page appears. Complete as many details as necessary. You can type all or part of a name into any field.
  3. Click . Your search results are displayed on the User Search page in a browse table.
  4. Select one of the records to display the details.
