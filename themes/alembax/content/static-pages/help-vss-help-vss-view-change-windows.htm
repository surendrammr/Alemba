Viewing Change Windows in Self Service Portal

Change Windows are designated time slots that have been arranged for the implementation of requests.

Before you start

The requests displayed depend on your permissions. You must have Review Own Requests, Review Org Requests and/or Review All Requests selected in the Options tab of your Self Service Portal role, depending on your viewing requirements.

Any linked config items must be enabled for viewing via the portal to view the details.

Any linked services must be enabled for viewing via the portal to view the details.

To view Change Windows for requests in Self Service Portal:

  1. On the My Options menu, select Requests > View Change Windows.
  2. The View Change Windows page appears with the name and description of each of the change calendars defined on your system.
  3. To view the change windows linked to a change calendar, select the change calendar name.
  4. The View Change Windows page appears with a list of all the linked change windows. For each change window, the Change Window ID, Start Date, and End Date are displayed. The unique system-generated identifier assigned to the change calendar is also displayed.
  5. To view the details of a change window, select the Change Window ID. The details of the selected change window are displayed in the Change Window Details page. The following details for the selected change window:
  • Unique system-generated identifier assigned the change window
  • Window ID
  • Change Calendar to which the change window is linked
  • Start and End Date/Time of the current change window
  • Linked Requests: A list of all requests to which this change window has been applied appears.
  • A list of all the config items linked to the selected change window
  • A list of all the services linked to the selected change window
