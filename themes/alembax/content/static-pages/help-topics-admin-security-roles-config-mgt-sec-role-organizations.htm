Configuration Management Security Roles: Organizations Tab

You can set permissions for Configuration Management Security Roles relating to Organizations and External Suppliers, including Object Management. This includes setting permissions for viewing, creating and updating organizations.Before you start

Before you start

You must have Security Roles setup selected in the Admin tab of your own General Access security role before you can assign or remove permissions for any security roles.

  1. Display the Configuration Management Security Roles details window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Security Roles, then select Configuration Management. All existing Configuration Management security roles are displayed.

    The pre-defined role All typically has all role options selected. However, some of the options are not selected by default and you should explore the permissions set before assigning the role.

    If you want to create a role, select and assign the role a name in the Role Name field. To view or update a role, select the role in the browse table and select .

  2. Assign the global permissions for the security role:

    3. Disable CMDB Security Analysts with this role have unlimited access to the CMDB, including the External Resource Search and Federated CMDB Admin options. Select this option to effectively turning off the granular security system for access to the CMDB. All CMDB items will open in edit mode, regardless of other permissions. Leave this deselected to assign specific permissions to the role.
    External Resource Search Analysts with this role can search for CMDB items from the External Resources Search window. Analysts with this permission are restricted to a Cached Resource Search so searches can only return CMDB details that have been imported and not excluded from the Federated CMDB. This option also allows Analysts to view external resource data via the Linked Resources option.
    Federated CMDB Admin

    Analysts with this role can import CMDB Items from the External Resource Search window. They can also perform a Full Resource Searchto search for Item Details within the CMDB which have not been imported into ASM Core, access the Federated CMDB Administration windows and access and process Discrepancy Reports through the CMDB Item Details Explorer option of the same name.

    This option is only enabled if External Resource Search is selected.
  1. Select the Organizations tab and make the necessary changes to the permissions.
Organization View Analysts with this role can search for and view organizations which are not flagged as External Suppliers. Selecting this will make the other options available.
Organization New Analysts with this role can create organizations which are not to be flagged as External Suppliers.

Organization View must be selected to enable this option.
Organization Update Analysts with this role can update details of organizations which are not flagged as External Suppliers.

Organization View must be selected to enable this option.
Organization Delete Analysts with this role can delete organizations which are not flagged as External Suppliers.

Organization View must be selected to enable this option.
External Supplier View Analysts with this role can search for and view Organizations which are flagged as External Suppliers. Selecting this will make the other options available.
External Supplier New Analysts with this role can create create organizations and flag them as External Suppliers.

External Supplier View must be selected to enable this option.
External Supplier Update Analysts with this role can flag organizations as External Suppliers, and update organizations which are flagged as External Suppliers.

External Supplier View must be selected to enable this option.
External Supplier Delete Analysts with this role can delete organizations which are flagged as External Suppliers.

External Supplier View must be selected to enable this option.
Organization Title Update Analysts with this role can modify the Title field for Organizations
Organization Cancel Lock This option cancels the lock set on Organizations. By default, only one Analyst can update an Organization record at any one time. Selecting this option disables this function and enables the Organization record to be updated by two or more Analysts simultaneously.
Organization Action by Default Analysts with this role can open organization details in edit mode directly. They do not need to use the Edit button.
Organization Object View Analysts with this role can download and view Objects that are attached to Organizations
Organization Object Update Analysts with this role can update Objects that are attached to Organizations.

Organization Object View must be selected to enable this option.
Organization Object Cancel Analysts with this role can cancel and ‘override’ a checkout of an object that is attached to an organization and restore it to its last checked-in state.

Organization Object Version Control must be selected in the CMDB Settings and Organization Object Update in this role must be selected to enable this option.

Organization Object Delete Analysts with this role can delete Objects that are attached to Organizations.

Organization Object View must be selected to enable this option.

Organization Stakeholder View

 Analysts with this role can view stakeholders that are linked to Organizations.

Organization Stakeholder Write

 Analysts with this role can update details for stakeholders that are linked to Organizations.

Organization Stakeholder View must be selected to enable this option.
Organization Cost Center View Analysts with this role can view or search for cost centers that are linked to Organizations
Organization Cost Center Update Analysts with this role can update cost centers that are linked to Organizations

Organization Cost Center View must be selected to enable this option.
  1. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so. Alternatively, select another tab, if appropriate.
