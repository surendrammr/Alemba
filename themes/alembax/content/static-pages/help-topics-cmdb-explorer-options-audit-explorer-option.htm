Audit Explorer Option

The Audit explorer option enables you to view an audit trail of the changes made to the details of the entity on screen.

Before you start

The relevant Audit Trail option should be selected. Depending on the item, the option is selected in the CMDB Settings window or the CMDBÂ Item Type Details window.

  1. From the details window, in the Explorer pane, expand Data, then select Audit.

    The <Entity> Audit Trail window appears, displaying details of changes. You can adjust the column widths if required.

  2. From here, you can copy details into another application by selecting them and then selecting .
  3. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.
