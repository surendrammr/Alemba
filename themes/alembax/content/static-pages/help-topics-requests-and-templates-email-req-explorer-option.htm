Email Explorer Option

The Email Explorer option enables you to send an email to the user of the request or to an analyst or group directly from the Request Details window.

 

Before you start

Notes can only be added to a request that is open. If you try to add a note to a request that is closed, ASM Core displays a warning message stating that the request cannot be updated.

 

  1. Select the Take Action button on the Request Details window if necessary to allow you to work on the request.
  2. In the Explorer pane, expand Communication, then select Email to display the Send Email window.
  3. The body of the email is already populated with the details of the request, and if objects are already attached to the request from which you are sending the message, these objects are listed in the top box. Complete any remaining details and send the email.

    4. You can also send an email to the user of the request when closing the request.
