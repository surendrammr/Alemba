System Partitions

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

You can create, view and delete partitions, as well as restoring partitions that have been deleted.

Creating a Partition

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

  1. Display the System Partitions window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand System, then select System Partitions.

  2. The System Partitions window appears with a list of all the partitions defined on your system .
  3. Select . A new row is added to the browse table.
  4. In the blank row, type the name of the partition you want to create.
  5. Repeat for every partition you want to create.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so.

Viewing Partitions

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

  1. Display the System Partitions window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    Select the System Partitions option from within the System group.

  2. The System Partitions window appears with a list of all the partitions defined on your system .
  3. Select Show Active List to view all of the active partitions, or Show Deleted List to show all deleted ones.

Deleting a Partition

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

You should be very cautious about deleting partitions, as it may affect the functioning of your system and your access to data. Consult Alemba® for more advice.

  1. Display the System Partitions window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand System, then select System Partitions.

  2. The System Partitions window appears with a list of all the partitions defined on your system.
  3. Select Show Active List to view all of the active partitions, if it is not already selected.
  4. Select the partition you want to delete by clicking in the relevant row.
  5. Select . ASM Core does not permanently delete partitions; it only marks them as deleted.
  6. Select to save the changes. Provide the Change Reasons if prompted to do so.

Restoring a Partition

You can restore deleted partitions, if necessary.

Before you start

You must have System Set Up selected in the Admin tab of your General Access security role.

  1. Display the System Partitions window, if it is not already on screen.

    Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand System, then select System Partitions.

  2. The System Partitions window appears with a list of all the partitions defined on your system.
  3. To view deleted partitions, select Show Deleted List. In this mode, is disabled.
  4. To restore a deleted partition,select the required partition, and select .
  5. Select to save the changes. Provide the Change Reasons if prompted to do so.
