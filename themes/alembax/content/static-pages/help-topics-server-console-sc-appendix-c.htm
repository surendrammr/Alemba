Server Console: ASM Core Registry Keys

To view a PDF version of the server console documentation, select the following link .

The following is a list of registry keys used in ASM Core. Registry keys are located in HKEY_LOCAL_MACHINE\SOFTWARE\VMWare\ASM Core.

Registry Key

Description

HKEY_LOCAL_MACHINE\SOFTWARE\alemba\ASM Core\SystemName

RemoveUnicode

When the value is set to '1' or 'Y' any text entered with Unicode characters will be removed and stored as normal text in the database.

EntityDataCacheSize

Comma-separated list of up to two values (for example, 1024, 16) to specify the cache size and amount to decrement entity data stored on the server to reduce database loads. The default and minimum values are: 128, 16. 

Note, this registry key should not be modified without first contacting Alemba® Support Services.
