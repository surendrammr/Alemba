Impact Criteria Explorer Option - CMDB Item

You can specify the criteria that will be used to apply an outage to a CMDB item when one or more CMDB items impacting this item are currently experiencing an outage.

Before you start

Track Availability must be selected for the CMDBÂ Item using the Availability option in the explorer pane.

  1. In the <Entity> Details window, in the Explorer pane, expand CMDB Item, then select Impact Criteria.
  2. The Impact Criteria window appears. Complete the details.
