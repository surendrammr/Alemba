Contract Support Type

You can define Contract Support Types to describe the type of support that is offered with a contract. For example you may wish to create gold, silver and bronze levels of support. You may define as many support types as you need.

Creating a Support Type

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select the Contract Support Type option from the Explorer pane. The Contract Support Type window appears, allowing you to define profiles for different types of contracts.
  1. Select . A new row is created to the browse table.
  2. Select the row and type the name of the new support type you want to create.
  3. Repeat for each support type you want to create.
  4. Select to save the changes. Provide the Change Reasons if prompted to do so.

Renaming a Support Type

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select the Contract Support Type option from the Explorer pane. The Contract Support Type window appears, allowing you to define profiles for different types of contracts.
  1. Select the support type entry you want to edit by selecting on it. The entry is highlighted.
  2. Type the name you want to assign for the selected support type.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.

Deleting a Support Type

Before you start

You must have CMDB Setup enabled within your General Access security role in order to configure any CMDB administration settings. Depending on if or how your system is partitioned, ensure that you are working in the correct partition.

You cannot reverse this procedure. If you delete a support type by mistake, you have to re-create the entry by creating a new support type.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand CMDB.

  2. Select the Contract Support Type option from the Explorer pane. The Contract Support Type window appears, allowing you to define profiles for different types of contracts.
  1. Select the support type entry you want to delete from the list.
  2. Select . If you attempt to delete the system default, you will receive a warning message.
  3. Select to save the changes. Provide the Change Reasons if prompted to do so.
