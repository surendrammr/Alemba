Linking Service Items to an Agreement

You can link and remove service items (that is, services, service actions, and service bundles) to and from the current agreement.Â 

When running an impact analysis against a service item, the linked agreements will be displayed in the Impact tab on the CMDB linking diagram for the service item, thus making it easy to identify the potential impacts of a period of unavailability of the service item.

Viewing Linked Service Items

Before you start

You must have Agreement Read selected in the Options tab of your Service Level Management security role.

  1. Display the Agreement Details window, if it is not already visible on screen, by searching for the agreement.
  2. In the Explorer pane, expand Agreement, then select Service Items.
  3. The Linked Service Items displays a browse table, listing all currently linked service items.
  4. Select the service item you want to view and then select Open to display the details window for the selected item.

Linking a Service Item

Before you start

You must have Agreement Update selected in the Options tab of your Service Level Management security role.

  1. Display the Agreement Details window, if it is not already visible on screen, by searching for the agreement.
  2. In the Explorer pane, expand Agreement, then select Service Items.
  3. The Linked Service Items displays a browse table, listing all currently linked service items.
  4. Select to make the buttons beneath the table active if necessary.
  5. Select . The CMDB Item Search window appears pre-selected for a service item search (that is, services, service actions, and service bundles). Search for and link the relevant service items.
  6. In the Link Type column, select the Link Type. The Link Type is a can be used to profile a linked service item.

  7. When a service item is linked to an agreement, the agreement also appears in the Agreements view on the corresponding service item.

Before you start

You must have Agreement Update selected in the Options tab of your Service Level Management security role.

  1. Display the Agreement Details window, if it is not already visible on screen, by searching for the agreement.
  2. In the Explorer pane, expand Agreement, then select Service Items.
  3. The Linked Service Items displays a browse table, listing all currently linked service items.
  4. Select to make the buttons beneath the table active if necessary.
