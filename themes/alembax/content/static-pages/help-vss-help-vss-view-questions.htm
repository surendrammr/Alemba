Viewing Your Questions

You can search for and view queries you have submitted.

Ask Questions must be enabled on vSelf Service Portal Settings window in ASM Core.

  1. On the My Options menu, select Self Help > View your Questions.

    The Your Questions page appears.

  2. Complete the details as required.

    3. Profile Use the tiered drop-down list to identify an article profile type. You can select from any of the tiers.
    Service Specify a service, using the button if you wish.
    Type Use the tiered drop-down list to identify a problem type. You can select from any of the tiers.
    Search within Use the options to select from current and past questions (which have received a response).
  3. Then select OK. Your questions are displayed in a summary format.
