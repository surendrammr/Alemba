Asking a Question in Self Service Portal

The Self Service Portal enables you to submit a query to a service desk operator.

The menu option will be displayed only if the Ask Questions option has been enabled on Self Service Portal Settings window in ASM Core.

  1. Select Self Help and then Ask a Question from the My Options menu. The Ask a Question page appears.
  2. Complete the details as required.

    3. Profile Use the tiered drop-down list to identify an article profile type. You can select from any of the tiers.
    Service Specify a service, using the button if you wish.
    Type Use the tiered drop-down list to identify a problem type. You can select from any of the tiers.
    Question Type your question, selecting the checkbox to receive an email copy of your question if you wish.
  3. Then select OK.
  4. A confirmation page displays your question and provides you with a call number with which to track it. You can review your query by selecting the View Your Questions menu option.
