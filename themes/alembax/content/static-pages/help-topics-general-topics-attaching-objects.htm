Attaching Objects

You can attach files to almost any part of the ASM Core application - calls, CMDB records, Knowledge Bank articles, emails and so on.

Before you start

You must have Object Update selected in the relevant security role.

  1. Display the <Entity> Details window if it is not already on screen.
  2. Select if necessary to activate the toolbar buttons and explorer options.

  3. In the Explorer pane, expand Data, then select Object.

    If there is already an object attached to the entity, Data and Object are displayed in orange text, with a small dot beside Object.

  4. The Attach Object to <Entity> window appears. From here, you can do any of the following, using the buttons on the toolbar:

    5. To return to the <Entity> Details window
    To upload a file
    To download a file

    To delete an attachment

    You must have Object Delete selected in your Bulletin Board security role to access this option.
  5. Select the button.
  6. From the Open dialog box, navigate to the folder containing the file, and select it before selecting Open.
  7. The Upload Object window appears. Here you can check that you have selected the correct file, add further description, using the HTML editor if available on your system, and select/deselect the Visible in Self Service Portal checkbox to indicate whether you want the attachment to be seen by users accessing the Self Service Portal. Make any changes you required, then select OK.

    8. Data and Object in the Explorer pane change color, although you may not be able to see this while Object is still selected.

  8. The file appears in the Upload/Download browse table.

    9. If a Windows Event Viewer file (*.evtx) is uploaded, you can view it by double-clicking on it. The file will be displayed in the Windows Event Viewer.

  9. Once the object has been uploaded, analysts with the appropriate permissions can do any of the following:

    10. Check Out To check the object out
    Check Out Exec To check out and execute the file
    Check In To check the file back in
    Cancel Check Out To cancel the check out

    You must have <Entity> Object Version Control selected in the <Entity> Settings window.
    View History To view the history of changes to the uploaded object
  10. Select the Main button to return to the <Entity> Details window.
  11. Continue adding/completing the details if necessary. Select to save the changes. Provide the Change Reasons if prompted to do so.
