Knowledge Search Explorer Option

The Knowledge Search explorer option enables you to search for Knowledge Bank articles that you can link to the current call.

Selecting this option opens the Knowledge Search window with the service, type and description automatically updated from the call.

The service linked to the call automatically populates the CMDB Item field in the Knowledge Search criteria panel.

You can specify additional criteria for your search. Once the search results are displayed, you can select one or more articles that you want to link to the current call.

While the call description is automatically updated on the Knowledge Search window (into the Search Text field), this field is not selected as a criterion by which to search. This is by design, since running a search with a large amount of words will take a long time and most likely return no results.

If you want to use the call description in the Knowledge Search, you should review the words listed and then select the Search Text field so that it is used.
