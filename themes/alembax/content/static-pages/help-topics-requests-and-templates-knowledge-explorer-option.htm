Knowledge Explorer Option

The Knowledge Explorer option enables you to search for Knowledge Bank entries and link them to the current request.

 

  1. Select the Take Action button on the Request Details window if necessary to allow you to work on the request.
  2. In the Explorer pane, expand Data, then select Knowledge to open the Linked Knowledge window with a list of any Knowledge Bank articles linked to the current request.

    3. When ASM Core inserts a Knowledge Bank entry to the current request, it inserts a “referral” or link to the location of the knowledge entry, rather than attaching the information (in the case of an object).

  3. To open a linked Knowledge Bank entry and view the entry, double-click the corresponding row in the browse table on the Linked Knowledge window.
  4. To link existing knowledge entries to the request, click the button. In the Knowledge Search window, specify the search criteria. In the list of matching results, click the button next to the title of the knowledge entry you wish to link to the request.
  5. As soon as a Knowledge Bank entry is linked to the current request, a small orange square appears next to the Knowledge Explorer option on the Request Details window.
  6. To unlink a knowledge entry linked to the current request, select the row in the browse table on the Linked Knowledge window and select . This will remove the referral to that entry from the current request.
  7. To create a knowledge entry from the Linked Knowledge window, select to display the New Knowledge Entry dialog box. Select the type of knowledge entry you wish to create and click OK. Specify the details, and click Save. The Linked Knowledge window appears with the entry you created, as it is automatically linked to the current request.
