Workflow Management Security Role: Forms Tab

You can set which forms Analysts can access, and specify particular types of access for each form.

Before you start

You must have Security Roles setup selected in the Admin tab of your own General Access security role before you can assign or remove permissions for any security roles.

To enable this tab, Forms must be selected in System Settings.

  1. Display the Workflow Management Security Roles details window, if it is not already on screen.

    Select â‰¡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Security Roles, then select Workflow Management. All existing Workflow Management security roles are displayed.

    The pre-defined role All typically has all role options selected. However, some of the options are not selected by default and you should explore the permissions set before assigning the role.

    If you want to create a role, select and assign the role a name in the Role Name field. To view or update a role, select the role in the browse table and select .

  2. Select the Forms tab . (You may need to scroll across to see this.)
  3. If you want to configure access to particular forms for this role, select the Forms Security option. If you do not select the option, the role will have unrestricted access to all change management forms.
  4. Having selected Form Security, select the screen set you want to make available to this role in the Select Form table and select Add.
  5. To set the access type for a particular form, click in the Access column for the form, and choose from Read Only to give Analysts with this role the ability to view the forms, but not edit them; or Read/Write to allow them to open and edit forms of the selected type.
  6. To remove a form, select it in the Selected Form table, then select Remove.
  7. Repeat until you have the form security set as you wish.
  8. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so. Alternatively, select another tab, if appropriate.
