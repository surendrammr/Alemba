Self Service Portal Role - CMDB Tab

The CMDB tab on the Self Service Portal Role window allows access to Services and Configuration Items in the Self Service Portal. You can filter this access based on subscriber group or stakeholder role.

Before you start

You must have Self Service Portal Setup enabled within your General Access Security Role in order to configure any Self Service administration settings.

  1. Select ≡ > Admin > System Administration.

    The System Administration window appears.

    In the Explorer pane, expand Self Service Portal.

    Select Self Service Portal Roles to open the window.

  2. Select to create a new role, or, if you want to update an existing role, select the role and then .
  3. If you are creating a new role, enter the name in the Role Name field.
  4. On the CMDB tab, select the permissions for the role:

    5. Services
    Review Own Services Select this option to enable Users with this role to search for and view Services that are assigned or linked to them.
    Review Org Services Select this option to enable Users with this role to search for and view Services that are assigned or linked to their organization.

    Review Own Services must be selected to enable this option.
    Review All Services Select this option to enable Users with this role to search for and view all Services in the Service Request Catalog which have been made visible to the Self Service Portal.

    Review Org Services must be selected to enable this option.
    Review Stakeholder Roles

    Select this option to limit the ability to search for and view all Services in the Service Request Catalog which have been made visible to the Self Service Portal to specific CMDB stakeholder roles. You can then select the stakeholder role that has portal visibility in the table below.

    Review All Services must be deselected to make this checkbox available.
    Service Request Catalog Select this option to enable Users with this role to search for, view and order service actions and service bundles through the Service Request Catalog.
    Service Request Catalog (Subscriber Group) Select this option to filter the Service Request Catalog by subscriber group. (You can make it available to all subscribers within the Service Action Details windows.)
    Service Request Catalog (Stakeholder Role) Select this option to filter the Service Request Catalog by Stakeholder role. (You can make it available to all stakeholders within the Service Action Details windows.)
    Catalog Filtered by Organization This option determines how the catalog will be filtered.

    Service Request Catalog Subscriber must be selected to enable this option.
    Configuration Items
    Review Own CIs Select this option to enable Users with this role to search for and view Configuration Items that are assigned or linked to them.
    Review Org CIs Select this option to enable Users with this role to search for and view Configuration Items that are assigned or linked to their organization.

    This option only enables users to review items assigned to the organization to which they are directly linked, not ones assigned to other organizations linked to their organization.

    Review Own CIs must be selected to enable this option.
    Review All CIs Select this option to enable Users with this role to search for and view Configuration Items in the system which have been made visible to the Self Service Portal.

    Review Org CIs must be selected to enable this option.
    User Search
    Review Org Users Select this option to enable Users with this role to search for and view details of Users linked to their organization.

    Users with this permission who are not linked to any organization cannot perform User searches.

    Review All Users Select this option to enable Users with this role to search for Users within and outside their organization.

    Users not linked to an organization can also search for Users if this option is enabled.

    Review Org Users must be selected to enable this option.
    Organization Search
    Review Org Organizations Select this option to enable Users with this role to search for and view organizations that are assigned or linked to their organization.
    Review All Organizations Select this option to enable Users with this role to search for and view all organizations to which they have access.

    Review Org Organizations must be selected to enable this option.

    Self Help Options
    Knowledge Bank Search Select this option to enable Users with this role to search for and view Knowledge Bank articles on the Self Service Portal.
    Scripts Select this option to enable Users with this role to access Scripts on the Self Service Portal through the Self Diagnosis menu.
    Self Administration Options
    Change Password Select this option to enable Users with this role to change their Portal password on the Self Service Portal.
    Change Details Select this option to enable Users with this role to change and update their details through the Self Service Portal.
    Delegation Setup Select this option to enable Users with this role to change the delegation of their tasks to another user through the Delegation Setup menu on the Self Service Portal.
    Analyst Search
    Analyst Search Select this option to enable Users with this role to search for Analysts, for example when filling out a request submission form containing an Analyst field.
    Group Search
    Group Search: Select this option to enable Users with this role to search for IPK or Workflow groups, for example when filling out a request submission form containing a group field.
    Location Search
    Review Org Locations Select this option to enable Users with this role to search for and view locations that are assigned or linked to their organization.

    Users who are not linked to an organization cannot perform location searches.
    Review All Locations Select this option to enable Users with this role to search for and view all locations to which they have access. Users who are not linked to an organization can perform location searches.

    Review Org Locations must be selected to enable this option.

    Chat  
    Chat User Select to allow users with this security role to participate in chats with analysts via the Self Service Portal.
  5. Select to save the changes and close the window. Provide the Change Reasons if prompted to do so. Alternatively, select another tab, if appropriate.
