Reopening a Closed Request

Once a request is closed, it is removed from the Requests Outstanding window. However, a request can be reopened if further action is required.

  1. Select â‰¡ and then Search. From the submenu, select Requests. The Request Search window appears.
  2. Select the Closed search filter in the search pane.
  3. Select any other filters you want to apply to the search.
  4. Click Search.
  5. Select the request you open to reopen in the Results browse table. The Request Details window opens. Select Re-Open at the bottom of the window.
