Call Search Explorer Option

The Call Search explorer option displays the Call Search window, where you can view all open calls logged against the CMDB item.

Viewing a call linked to the CMDB Item

  1. From the explorer pane in the Details window, select the Call Search option. This normally appears within the Linked Resources group. Scroll up to locate the group, and expand it if necessary.
  2. The Call Search window opens. If there are open calls raised for the CMDB item, these display automatically in the browse table in the search results pane. You can adjust the column widths if required.
  3. To view the details of a linked call, select the call in the search results pane and select .
