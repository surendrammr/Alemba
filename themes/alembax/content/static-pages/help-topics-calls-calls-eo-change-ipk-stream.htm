Change IPK Stream Explorer Option

You can change the stream of a call after it has been logged. For example, you may wish to change the call stream from ‘Hardware’ to ‘Software’ if these streams have been defined by your system administrator.

This option is only available if IPK tiers are enabled by your system administrator in the IPK Settings and you have access to more than one stream. You can only select a stream to which you have access in the IPK Streams tab of your IPK Management security role.

Changing the stream of a call may change the custom screen set that has been used to log the call.

  1. Display the Call Details window if it is not already on screen.
  2. In the explorer pane, expand Workflow, then select Change IPK Stream.
  3. In the IPK Stream dialog box, select the required stream you wish to apply.
  4. Click OK.

The screen refreshes and displays the window associated with the selected stream. The new stream of the call appears in the Information panel.
