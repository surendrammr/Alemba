Searching for Items

You may not always know the exact call, person or CMDB item you need, and you can search for items on most input fields on Details, Workload and Outstanding windows in a number of ways.

You can search using:

  • The search function (see Using the search function on page 1)
  • Multi-tier selectors, for items that have values organized in multiple tiers
  • Search windows
  • Outstanding Item searches

If you find that you regularly carry out the same search, that is, you always specify the same search criteria, you can save the search, so that you can run it again as required, and retrieve the latest data.

You can share saved searches with other analysts, who might find them useful, and view searches that other analysts have shared with your IPK or Workflow group.
