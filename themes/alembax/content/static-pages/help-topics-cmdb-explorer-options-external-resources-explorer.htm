External Resources Explorer Option

The External Resources explorer option enables you to view external resources such as directory server users linked to the current entity. You can also link and unlink them.

Viewing External Resources

Before you start

You must have External Resource Search or Federated CMDB Admin selected in the opening window of your configuration management security role.

  1. From the <Entity> Details window, in the Explorer pane, expand Federated CMDB, then select External Resources.

    2. If there are external resources linked to the current entity, this option is shown in orange and preceded by an orange square.

  2. The linked resources are displayed. Select one to view the information in more detail.
  3. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.

Linking an External Resource to an Entity

Before you start

You must have External Resource Search or Federated CMDB Admin selected in the opening window of your configuration management security role.

You can link an external resource to an entity even if the details are being viewed by another analyst

  1. From the <Entity> Details window, in the Explorer pane, expand Federated CMDB, then select External Resources.

    2. If there are external resources linked to the current entity, this option is shown in orange and preceded by an orange square.

  2. The linked resources are displayed.
  3. Select the button to display the External Resource Search window. Complete the search criteria in the Federated CMDBÂ Search Criteria pane, and select Search.
  4. Select the resource you want to make available to the person and select .
  5. The External Resources window appears once more, with the new resource listed.
  6. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.

Unlinking an External Resource

Before you start

You must have External Resource Search or Federated CMDB Admin selected in the opening window of your configuration management security role.

  1. From the <Entity> Details window, in the Explorer pane, expand Federated CMDB, then select External Resources.
  2. The linked resources are displayed. Select the resource you want to unlink.

  4. Select from the toolbar or Return To Main at the top of the Explorer pane to return to the Details window.
