Viewing Integration Schedule Activity

You can view the details of scheduled scan which has been run against a configured source.

Before you start

You must have Integration Setup selected in your General Access Security Role.

  1. Select â‰¡ > Admin > Integration.

    The Integration Platform Settings window appears.

  2. Select the Activity option from the explorer pane .
  3. Complete the details:

    4. From and To

    Select the start and end date to define the period in which you want to view activity.

    Connector

    select the connector for which you want to view activity

    Source

    select the source for which you want to view activity
  4. Select the Scheduling tab to view all scans run in the period.
  5. To view the activity details about a scan, double-click the entry in the browse table. This displays the Integration Schedule Execution window.

    6. The top part of the window displays the scheduling details defined when the schedule was set up, as well as the overall status of the scan.

    Schedule The name of the schedule defined
    Scheduled Start Date The start date of the schedule
    Server The name of the source on which the scan was run
    Last Updated The last time the scan was run
    Scan Type

    For integrations with a directory server, indicates the nature of the scan that took place.

    Full A scan was completed of every analyst or user imported into ASM Core from a directory server, regardless of whether their information has been updated.
    Incremental

    A scan of analyst or user details that had been updated, or new records for new analysts or users from the directory server.
    Full then Incremental A full scan was run initially, followed by incremental scans

    Status Indicates whether the scan was completed successfully
    Message Provides more information if a scan did not complete successfully

    There are two views below:

    Statistics This tab displays a statistical summary of the number of sources, resource types, link types, scanned, as well as the number of resources, links created, updated, deleted, or restored during the scan.
    Synchronization This tab displays the log for the scan which includes warnings and error messages. You can double-click an entry to view the log details
  6. Click the button to update the activity details if needed. This is useful if a scheduled scan is currently running and you wish to see the new results.
