Introduction

You can use the ASM Server Console application to create, rename, delete and configure your ASM Systems. You can also use the Server Console to perform a range of tasks, such as archiving calls, and importing/exporting workflow templates.  

The following topics will take you through how to access the server console, and create and maintain your system.

  • Accessing the Server Console
  • Creating a new system
  • Configuring your systems
  • Server Console system tasks
